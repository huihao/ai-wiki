# Distillation

## Definition

In the context of large language models, distillation refers to training smaller "student" models on outputs (or reasoning traces) generated by larger "teacher" models. Unlike classical knowledge distillation (which uses teacher logits), LLM distillation typically involves supervised fine-tuning on high-quality teacher-generated data.

## Evolution

- **2015:** Hinton et al. introduced knowledge distillation
- **2024-2025:** LLM distillation became a primary method for creating efficient reasoning models
- **Key finding:** Distillation is far more effective than pure RL for smaller models
- **Current:** Used to create models like DeepSeek-R1-Distill that run on consumer hardware

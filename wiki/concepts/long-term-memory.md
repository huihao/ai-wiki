# Long-Term Memory (LTM)

## Definition

Persistent memory storing past experiences, learned knowledge, and historical information for long-term agent use. Often external to immediate context.

## Characteristics

- Persistent storage
- Large capacity
- External to working context
- Requires retrieval mechanisms
- Enables learning over time

## Integration with STM

In unified memory systems like AgeMem, LTM is combined with short-term memory inside the agent, with the agent actively managing what to store, retrieve, summarize, or discard.

## Storage Approaches

- External databases (Memex)
- Key-value Memory Banks (UMA)
- Episodic storage (MemRL)
- Hierarchical systems (Pancake)

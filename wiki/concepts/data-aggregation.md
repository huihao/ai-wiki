# Data Aggregation

## Definition

Data aggregation is the process of collecting, combining, and summarizing data from multiple heterogeneous sources into a unified representation, enabling cross-source analysis, knowledge integration, and population-scale research. Projects like Data Commons demonstrate this at scale by unifying data from thousands of sources under a common schema.

## Key Aspects

- Combines data from diverse sources (government databases, research institutions, public APIs) into a single queryable system
- Requires schema mapping and normalization to reconcile different data formats, ontologies, and measurement units
- Enables cross-domain queries that would be impossible with siloed data (e.g., correlating health outcomes with economic indicators)
- Population-scale datasets (billions of data points) require distributed storage and compute infrastructure
- Data Commons projects aim to make aggregated data freely accessible via APIs and common data models
- Quality challenges include handling missing data, resolving entity conflicts, and maintaining provenance tracking
- Enables downstream applications like machine learning, policy analysis, and scientific research

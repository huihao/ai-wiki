# Complex Domain

## Definition

In the Cynefin framework, a Complex domain is one where cause-and-effect relationships can only be understood in retrospect — they are not predictable in advance. This domain requires safe-to-fail experimentation rather than analysis, as small changes can produce disproportionately different results. GenAI-based coding agents are argued to belong in this domain.

## Key Aspects

- **Unpredictable Outcomes**: The relationship between prompt and code output cannot be fully predicted
- **Emergent Behavior**: Same prompt can yield different outputs due to probabilistic nature
- **Experimentation Required**: Cannot standardize through "one true way"; must iterate and observe
- **Rapid Feedback**: Need fast cycles of action and observation to learn what works
- **Safe-to-Fail**: Design experiments so failure is recoverable, not catastrophic
- **Non-Linear Adoption**: Success emerges through iteration rather than following a plan
- **Governed by Feedback Loops**: Not by defined rules (as in complicated domains)

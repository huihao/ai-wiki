# Agentic Memory

## Definition

Memory architecture where memory management becomes part of the agent's decision-making process. Agents actively decide what to store, retrieve, summarize, or discard.

## Key Features

- Unified short-term and long-term memory
- Agent-controlled memory operations
- Decision-making includes memory management
- Reinforcement learning improves performance
- Better memory efficiency on long tasks

## Architecture Types

### AgeMem (Agentic Memory)

Framework unifying STM and LTM inside the agent, making memory management part of decision-making.

### UMA (Unified Memory Agent)

RL-trained agent with dual memory system: compact global summary plus structured key-value Memory Bank supporting CRUD operations.

## Benefits

- Improved long-horizon reasoning
- Better state tracking
- Adaptive memory management
- Efficient use of context

## Learning Approaches

- Reinforcement learning for memory decisions
- Training on long tasks
- Performance and efficiency optimization

# Learning Roadmap

Structured curriculum or guide for systematically acquiring knowledge and skills in a domain.

## Purpose

- Clear progression path
- Identified prerequisite knowledge
- Recommended resources
- Practical projects for application
- Avoiding "tutorial hell"

## LLM Learning Roadmap Structure

### Phase-Based Approach

**Phase 0**: Foundations
- Math prerequisites (linear algebra, probability)
- Programming basics (Python, PyTorch)
- First project: Build Micrograd

**Phase 1**: Core Concepts
- Transformer architecture
- Attention mechanisms
- Implementation project: Mini-GPT

**Phase 2**: Scaling
- Distributed training
- Scaling laws
- Parallelism strategies

**Phase 3**: Advanced Topics
- Alignment methods (RLHF)
- Fine-tuning (LoRA)
- Production deployment

**Phase 4**: Production
- Inference optimization
- Quantization
- Deployment strategies

## Learning Resource Types

1. **Visual Intuition**: Video explanations (3Blue1Brown, Karpathy)
2. **Formal Theory**: Academic lectures (Stanford, MIT)
3. **Papers**: Primary research sources
4. **Coding Projects**: Hands-on implementation

## Best Practices

- Build mental models before deep theory
- Implement from scratch to understand internals
- Use papers after establishing intuition
- Focus on one domain, avoid breadth-first learning
- Complete projects, not just consume content

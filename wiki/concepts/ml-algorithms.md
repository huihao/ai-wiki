# ML Algorithms

## Definition

Machine learning algorithms are computational procedures that learn patterns from data to make predictions or decisions without being explicitly programmed. They span optimization methods, statistical models, and neural architectures, forming the foundational toolkit for building intelligent systems across supervised, unsupervised, and reinforcement learning paradigms.

## Key Aspects

- **Implementation-based learning**: Platforms like TensorTonic emphasize learning ML by coding 200+ algorithms from scratch rather than reading tutorials, covering gradients through transformers
- **Core algorithm categories** include optimization (Adam, SGD), loss functions (cross-entropy), normalization (batch normalization), and architectural building blocks (softmax, attention)
- **Research track implementations**: Building state-of-the-art architectures from research papers using a paper-to-code approach
- **Interactive visualization**: Mathematical foundations explored through interactive tools, making abstract concepts concrete
- **Gamification drives engagement**: Leaderboards, contribution streaks, and badge systems for categories like Transformers, Neural Networks, and Optimization
- **Zero-setup browser execution**: Cloud-native runners allow instant feedback without local environment configuration
- **Key algorithms by domain**: Gradient descent variants, attention mechanisms, normalization layers, and activation functions form the essential vocabulary of modern ML

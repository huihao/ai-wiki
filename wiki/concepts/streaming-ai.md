# Streaming AI

## Definition

Streaming AI refers to the pattern of delivering AI model responses in real-time chunks rather than waiting for complete responses. This enables progressive rendering, improved user experience, and reduced latency perception in AI applications.

## Evolution

Streaming AI responses address the latency problem inherent in large language model generation. By chunking responses progressively, applications can render text incrementally, improving perceived performance. This pattern has become standard in modern AI SDKs and chat applications.

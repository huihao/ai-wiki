# Cynefin

## Definition
The Cynefin framework is a sense-making and decision-making model that classifies problems into five domains (Clear, Complicated, Complex, Chaotic, and Disorder) based on the relationship between cause and effect. Each domain prescribes a different approach to management and decision-making, making it a powerful tool for understanding why different situations require fundamentally different strategies.

## Key Aspects
- Clear domain: cause and effect are obvious; apply established best practices
- Complicated domain: cause and effect are knowable through analysis and expertise; use good practices
- Complex domain: cause and effect are only understood in retrospect; requires safe-to-fail experimentation
- Chaotic domain: no perceivable cause-effect relationship; act first, then sense
- Using GenAI coding agents is a complex domain because small prompt changes produce disproportionately different outcomes
- Traditional development tools operate in clear/complicated domains, but agent-based development requires complex-domain approaches
- Organizations must simultaneously manage the complexity of software systems and the complexity of using agents
- Success in complex domains emerges through iteration, not top-down planning

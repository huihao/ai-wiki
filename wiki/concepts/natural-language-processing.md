# Natural Language Processing

## Definition

Natural Language Processing (NLP) is a field at the intersection of computer science, artificial intelligence, and linguistics focused on enabling computers to understand, interpret, generate, and manipulate human language. Core NLP tasks include tokenization, part-of-speech tagging, named entity recognition, parsing, sentiment analysis, machine translation, question answering, and text generation. Modern NLP is dominated by [[transformers|transformer-based]] [[large-language-models]] that are pre-trained on massive text corpora and adapted to specific tasks through fine-tuning or prompting.

## Key Proponents / Critics

- [[daniel-jurafsky]] — co-author of "Speech and Language Processing," the standard NLP textbook
- [[james-h-martin]] — co-author of "Speech and Language Processing" with Jurafsky

## Related Concepts

- [[transformers]] — the architecture that revolutionized NLP starting in 2017
- [[bert]] — bidirectional pre-trained model that set new standards on NLP benchmarks
- [[gpt]] — autoregressive language model family that demonstrated scaling laws in NLP
- [[word-embeddings]] — dense vector representations that replaced sparse features in NLP
- [[word2vec]] — the influential word embedding model that launched the neural NLP era
- [[attention-mechanisms]] — enabled models to learn long-range dependencies in text
- [[machine-translation]] — one of the original and most impactful NLP applications
- Sentiment Analysis — classification of text by emotional tone, a common NLP task
- [[named-entity-recognition|NER]] — identifying and classifying named entities in text
- [[text-generation]] — producing coherent text, the core capability of modern LLMs
- [[tokenization]] — the process of segmenting text into tokens for model input
- [[information-retrieval]] — finding relevant documents, closely related to NLP applications

## Evolution

- **1950s**: Alan Turing proposes the Turing Test; early machine translation efforts begin
- **1960s-1970s**: Rule-based systems and hand-crafted grammars dominate NLP
- **1980s-1990s**: Statistical methods (n-grams, HMMs, CRFs) replace rule-based approaches
- **2003**: Bengio et al. introduce neural language models with distributed word representations
- **2013**: [[word2vec]] makes word embeddings practical and widely adopted
- **2014**: Sequence-to-sequence models with attention improve machine translation
- **2017**: [[transformers]] introduced, fundamentally changing NLP
- **2018**: [[bert]] achieves state-of-the-art on 11 NLP tasks; GPT-1 demonstrates pre-training at scale
- **2020**: GPT-3 shows emergent [[few-shot-learning]] abilities in NLP
- **2022-2025**: ChatGPT, Claude, and Gemini make NLP-based systems mainstream consumer products

# Open Source Agents

## Definition

Open source agents are AI coding assistants whose source code is publicly available, enabling community inspection, customization, and extension, in contrast to proprietary tools where the implementation details and behavior are controlled by a single vendor.

## Key Aspects

- **Transparency and Auditability**: Open source agents allow developers to inspect exactly how tools are invoked, how context is managed, and what system prompts guide behavior, eliminating the "black box" problem of proprietary alternatives
- **Community-Driven Extension**: Public repositories enable community contributions of features, bug fixes, and integrations, accelerating development and broadening platform support
- **Customization**: Teams can fork, modify, and tailor agent behavior to their specific workflows, coding conventions, and security requirements without vendor lock-in
- **MCP Marketplace Integration**: Projects like Cline demonstrate open agents connecting to MCP tool marketplaces, enabling extensible tool ecosystems that users can browse and install
- **Competitive Performance**: Open source agents like pi (minimal coding agent) demonstrate that minimal, transparent designs can match or approach the performance of complex proprietary systems on benchmarks like Terminal-Bench

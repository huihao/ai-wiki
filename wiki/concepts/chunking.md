# Chunking

## Definition

In the SOAR cognitive architecture, chunking is the automatic creation of new production rules (procedural knowledge) from successful problem-solving episodes. When SOAR reaches an impasse and resolves it, the resulting problem-solving trace is compiled into a new production rule that can be directly applied to similar situations in the future.

## Key Aspects

- Automatic compilation of experiences into reusable rules
- Bridges episodic experience with procedural knowledge
- Enables learning without explicit programming
- Analogous to how modern agents learn from past task completions

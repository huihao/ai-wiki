# ML Engineering

Engineering discipline focused on building, deploying, and maintaining machine learning systems in production.

## Role Definition

Hybrid of research and engineering:
- **Research focus**: Model architecture, algorithms, experimentation
- **Engineering focus**: Systems, deployment, monitoring, scalability
- **Research Engineer**: Both model development and production infrastructure

## Key Skills

### Machine Learning
- Model architecture understanding
- Training pipeline design
- Hyperparameter tuning
- Evaluation metrics
- Data preprocessing

### Software Engineering
- Production code quality
- Distributed systems
- Performance optimization
- Testing and debugging
- Version control and CI/CD

### Systems Engineering
- GPU/TPU infrastructure
- Model serving and inference
- Monitoring and observability
- Resource management
- Scaling strategies

## Career Trajectory

Based on [[max-mynter-research-engineer-career|Max Mynter's journey]]:

### Entry Path
- Self-taught or formal ML education
- Portfolio projects demonstrating end-to-end systems
- Open-source contributions to ML infrastructure
- Publications or research projects

### Growth Path
- Technical leadership (architect role)
- Specialization (training systems, inference optimization, specific domains)
- Staying individual contributor vs. management track
- Building rare skills in frontier areas (foundation models, agents)

## Preparation Strategy

### Skill Gaps
- **LLM experience**: Build projects with language models, TinyStories-style experiments
- **CS fundamentals**: Learn systems programming (Rust/C++), contribute to production codebases
- **Publications**: Rework thesis or research into peer-reviewed submissions

### Portfolio Projects
- Complete end-to-end ML systems (data → model → deployment)
- Open-source contributions to high-visibility repos (ruff, uv)
- Presentations at meetups/conferences (AI Tinkerers)
- Research publications (any peer-reviewed venue)

## Industry Roles

### Foundation Model Labs
- Mistral AI, OpenAI, Anthropic, Google DeepMind
- Billion-dollar funding, frontier research
- Highly competitive, require both research and systems skills

### Applied ML Companies
- Scale-ups with ML infrastructure teams
- Startups with ML-powered products
- Established tech companies with ML platforms

# Agent Learning

## Definition

Agent learning refers to the mechanisms by which AI agents improve their performance over time through experience, feedback, and self-reflection. Rooted in reinforcement learning principles, it encompasses how agents learn from their successes and failures to make better decisions in future interactions.

## Key Aspects

- Rich Sutton's foundational reading list emphasizes the RL approach to AI: learning through reward signals from interaction with an environment
- Self-improving agents track patterns in their mistakes and adjust strategies accordingly
- Learning from error observations provides correction signals that guide future behavior
- The "bitter lesson" suggests that general methods that leverage computation tend to outperform human-designed approaches
- Agent learning operates across multiple timescales: within-session adaptation, cross-session memory, and long-term strategy evolution
- Balancing exploration (trying new approaches) with exploitation (using known successful strategies) remains a core challenge

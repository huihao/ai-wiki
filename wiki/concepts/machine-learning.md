# Machine Learning

## Definition

Machine learning (ML) is a subfield of artificial intelligence focused on building systems that learn from data rather than following explicitly programmed rules. ML encompasses supervised learning (predicting labels from labeled data), unsupervised learning (discovering structure in unlabeled data), and reinforcement learning (learning from environmental feedback). Modern AI engineering is built on decades of ML research in optimization, statistics, and representation learning.

## Key Proponents / Critics

- **Foundational figures**: Arthur Samuel (coined "machine learning", 1959), Tom Mitchell (formal definition), Vapnik (statistical learning theory)
- **Modern era**: Hinton, LeCun, Bengio (deep learning pioneers); [[ilya-sutskever]], [[andrej-karpathy]] (modern LLM era)

## Evolution

- **1950s–1990s**: Rule-based systems, statistical methods, early neural networks
- **2000s**: Kernel methods, SVMs, random forests, boosting
- **2012**: AlexNet victory sparks the deep learning revolution
- **2017–2020**: Transformers, BERT, GPT-2 shift NLP to pre-training
- **2022–2024**: LLMs become the dominant ML paradigm; generative AI mainstream
- **2025–2026**: Agentic AI, reasoning models, and AI engineering emerge as new frontiers

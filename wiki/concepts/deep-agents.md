# Deep agents

## Definition

LangChain's agent harness abstraction that provides context engineering capabilities (summarization, tool offloading, planning) and supports a virtual filesystem with pluggable storage. Deep Agents prioritizes the memory system (AGENTS.md, skills, subagents, tools.json) and targets citizen developers for specialized workflow automation.

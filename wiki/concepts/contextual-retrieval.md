# Contextual Retrieval

## Definition

Contextual Retrieval is a technique that improves vector-based search by prepending document-level context to each text chunk before embedding, ensuring that chunk representations carry awareness of the broader document they belong to. It combines this contextual enrichment with hybrid search (combining semantic embeddings and TF-IDF) to significantly reduce retrieval failure rates.

## Key Aspects

- Chunks are enriched with a brief document-level summary generated by an LLM before embedding, preventing loss of document-wide context
- Uses hybrid search combining LLM-based embeddings and TF-IDF vectors, storing both representations for each chunk
- Retrieval pipeline: embed query, run both ANN and TF-IDF search, apply rank fusion, rerank results, generate answer
- Developed by Anthropic and demonstrated to cut retrieval failure rates from approximately 5% down to approximately 0.5%
- Addresses a fundamental limitation of naive RAG: chunks lose the context of the larger document they come from
- Particularly effective for documents where meaning depends on document-level context rather than local context alone

# Agent Capabilities

## Definition

Agent capabilities refer to the range of tools, functions, and actions that an AI agent can perform. In the context of AI coding agents, capabilities are shaped by the underlying model, the tools registered in the system, and the framework's design for tool discovery and invocation.

## Key Aspects

- The AI coding tool ecosystem has democratized access to top-tier models, making professional-grade capabilities available at low cost
- Tools define what agents can do — file operations, code execution, web search, API calls
- Different models provide different capability tiers (e.g., GLM 4.5, Qwen3 Coder, Claude, GPT-5)
- Agent capabilities can be extended through MCP (Model Context Protocol) tool servers
- Open platforms like OpenCode.ai offer multi-model access with agent setup using fresh context per subtask
- Capabilities span code generation, debugging, testing, deployment, and project management

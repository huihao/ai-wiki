# Statistical Analysis

## Definition

Statistical analysis is the systematic process of collecting, examining, summarizing, and interpreting quantitative data to identify patterns, test hypotheses, and support decision-making. In the context of machine learning and data science, statistical analysis underpins model evaluation, feature selection, and the validation of experimental results, providing the mathematical framework for distinguishing signal from noise in datasets.

## Key Aspects

- **Descriptive statistics**: Summarize data through measures of central tendency (mean, median, mode), dispersion (variance, standard deviation), and distribution shape (skewness, kurtosis) to provide an overview of dataset characteristics.
- **Inferential statistics**: Draw conclusions about populations from samples using hypothesis testing (t-tests, chi-square tests, ANOVA), confidence intervals, and p-values to determine whether observed patterns are statistically significant.
- **Probability distributions**: Model data using theoretical distributions (normal, Poisson, binomial, exponential) to make predictions and quantify uncertainty in measurements and model outputs.
- **Regression analysis**: Examine relationships between variables using linear, logistic, and nonlinear regression to predict outcomes and understand factor contributions.
- **Correlation and causation**: Distinguish between statistical association (correlation coefficients) and causal relationships, which require controlled experiments or causal inference methods.
- **Statistical significance in ML**: Used to compare model performance (e.g., paired t-tests across cross-validation folds), assess feature importance, and validate that improvements are not due to random variation.

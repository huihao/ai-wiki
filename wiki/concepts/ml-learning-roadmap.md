# ML Learning Roadmap

## Definition

A structured progression from foundational mathematics through classical machine learning to deep learning and cutting-edge research. Emphasizes that each stage builds on the previous: foundational math provides tools, classical ML provides intuition patterns, deep learning provides architectures, and cutting-edge ML requires reading research papers bottom-up.

## Key Details

- **Stage 1 — Foundational Math**: Algebra, calculus, linear algebra, probability/statistics (~70 hours for someone with single-variable calc background)
- **Stage 2 — Classical ML**: Code models from scratch (linear regression → small neural networks). Builds pattern recognition for loss functions, gradient descent, model architecture.
- **Stage 3 — Deep Learning**: Multi-layer networks with task-specific architectures. Requires Stage 2 patterns to avoid confusion.
- **Stage 4 — Cutting-Edge ML**: Transformers, LLMs, diffusion models. Read from basics up, not top-down. Literature search → foundational papers → latest research.
- **Key insight**: Classical ML knowledge gives intuition for modern papers — equations become recognizable patterns rather than hieroglyphics.

# AI Chatbot

## Definition

An AI chatbot is an interactive application that uses large language models to engage in natural language conversations with users, providing responses, answering questions, and performing tasks through a conversational interface. Modern chatbots extend beyond simple Q&A to support streaming responses, tool use, and multi-turn context management.

## Key Aspects

- The AI SDK by Vercel provides starter templates specifically for building AI chatbots with streaming support
- Modern chatbots support real-time streaming of responses rather than waiting for complete answers
- Multi-model fallbacks allow chatbots to switch between different LLM providers for reliability
- Built-in chatbot examples demonstrate patterns for message history, context management, and error handling
- Production chatbots require careful prompt caching to manage costs effectively
- The shift from simple chatbots to agents represents increasing autonomy and tool integration

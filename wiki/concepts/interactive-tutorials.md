# Interactive Tutorials

## Definition

Interactive tutorials are educational web-based guides that teach concepts through manipulable visualizations, step-by-step demonstrations, and hands-on experiments. Red Blob Games by Amit Patel is the archetypal example, offering free, ad-free interactive explanations of math, algorithms, and game development concepts using D3.js and Vue.js visualizations that readers can drag, adjust, and explore.

## Key Aspects

- **Red Blob Games**: Amit Patel's website (active since 1995, interactive since 2007) covering pathfinding, graph theory, probability, geometry, and procedural generation
- **Technology Stack**: Vue.js tutorials (2015+) with drag-and-drop interactions; D3.js (2011-2015); Canvas, KaTeX, pointer events for rich interactivity
- **Topics Covered**: Tower defense pathfinding (BFS, Dijkstra, flow fields), hex grids, 2D visibility, Bezier curves, probability distributions, polygonal map generation
- **Design Principles**: Start with motivating examples from games; build intuition through manipulation before formal definitions; make math tangible
- **Open Source**: MIT or Apache v2 licensed; commercial use allowed; starter code available for React, Vue, lit-html, vanilla JS, Preact
- **Responsiveness**: Interactive diagrams designed to work on mobile with touch events, not just desktop mouse
- **Diagram-Text Highlighting**: Synchronized highlighting between diagrams and explanatory text helps readers connect visual and verbal explanations
- **Community Influence**: Inspired by 3Blue1Brown, Mathologer, and John Carlos Baez; collaborated with Peter Norvig on AI textbook diagrams

# Graph Knowledge Base

## Definition

A Graph Knowledge Base is a knowledge representation system that organizes information as entities (nodes) and their relationships (edges), enabling multi-hop reasoning and complex queries over interconnected data. Unlike traditional vector-only databases, graph knowledge bases capture semantic relationships explicitly.

## Key Characteristics

- Entities as nodes with properties and descriptions
- Relationships as edges connecting entities with semantic meaning
- Enables multi-hop reasoning (traversing relationship chains)
- Supports complex queries combining vector search with graph traversal
- Better for questions requiring understanding of connections

## Evolution

Graph knowledge bases have evolved from simple entity-relationship models to sophisticated systems combining semantic embeddings, automatic extraction, and multi-hop reasoning. Modern systems like R2R integrate graphs with vector search for hybrid retrieval strategies.

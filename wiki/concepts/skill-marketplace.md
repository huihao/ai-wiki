# Skill Marketplace

## Definition

A skill marketplace is a platform for discovering, publishing, and installing agent skills—modular capability packages for AI agents. It functions like a package manager specialized for agent extensibility.

## Key Proponents / Critics

- skills.sh — Primary marketplace platform
- [[microsoft|Microsoft]] — Major contributor
- [[firebase|Firebase]] — Skills provider

## Related Concepts

- [[agent-skill|Agent Skills]] — Packages distributed through marketplace
- [[agent-extensibility|Agent Extensibility]] — Problem solved by marketplace

## Evolution

Skill marketplaces represent an emerging pattern for agent capability distribution, analogous to npm for JavaScript or pip for Python but specialized for AI agents. The leaderboard format with installation metrics suggests quality signaling through adoption. Major providers (Microsoft) dominate current distribution, indicating potential platform consolidation.

## Practical Implications

Marketplaces enable standardized discovery and installation of agent capabilities. Installation metrics provide quality signals for skill selection. The centralized platform approach simplifies skill management compared to scattered repositories. Potential challenge: ensuring skill compatibility and security across diverse agent frameworks.

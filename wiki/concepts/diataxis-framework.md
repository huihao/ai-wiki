# Diátaxis Framework

## Metadata

- **Type**: Documentation Framework
- **Creator**: Daniele Procida
- **URL**: https://diataxis.fr
- **Purpose**: Technical writing organization

## Definition

The Diátaxis framework is a systematic approach to technical documentation that divides content into four distinct modes: tutorials, how-to guides, reference, and explanation. Each mode serves a different user need and requires different writing approaches.

## Four Modes

### Tutorials
- Learning-oriented.
- Guided steps for beginners.
- Safe environment for practice.
- Focus on confidence-building.

### How-to Guides
- Problem-oriented.
- Practical steps to achieve goals.
- Assume some knowledge.
- Focus on real-world tasks.

### Reference
- Information-oriented.
- Technical descriptions.
- Accurate and complete.
- Structured for lookup.

### Explanation
- Understanding-oriented.
- Context and background.
- Clarifies why things work.
- Connects concepts.

## Benefits

- Clear separation of user needs.
- Consistent documentation structure.
- Better user navigation.
- Avoids mixing purposes.

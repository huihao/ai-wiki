# Modularity

## Definition

The degree to which a system's components may be separated and recombined. In software engineering, modularity means designing systems with clean interfaces between independent modules, so changes to one module don't cascade to others. High modularity reduces [[coupling-complexity]] by isolating concerns — a key enabler for [[heuristic-learning]] where coding agents must maintain growing Heuristic Systems.

## Related Concepts

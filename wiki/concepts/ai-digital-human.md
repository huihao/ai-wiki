# AI Digital Human

## Definition

AI digital humans refer to AI-powered systems that simulate human-like interaction capabilities for complex task processing in operational environments. In practice at companies like Taobao, they embody a three-component framework — perception (AI Eyes), decision (AI Brain), and action (AI Hands) — to automate complex repetitive workflows that previously required human operators.

## Key Aspects

- Three-component architecture: AI Eyes (perception via SDK integrations), AI Brain (decision making), AI Hands (action execution via API calls)
- Two decision modes: Workflow Mode (fixed sequence for predictable tasks) and Agent Mode (dynamic planning for adaptive tasks)
- Three automation patterns: Tool Assistant (single tasks), Complex Workflow (multi-step serial processes), Intelligent Agent (autonomous planning)
- Core philosophy: any repetitive task with clear inputs, logical processes, and fixed actions can be delegated to AI
- Handles both explicit repetition (batch labeling) and implicit repetition (data governance, ticket troubleshooting)
- Real-world applications include ticket processing, data governance, and baseline operations at scale

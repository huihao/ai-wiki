# IDE Integration

## Definition

IDE integration refers to the direct embedding of AI programming assistants into development environments such as VS Code and JetBrains IDEs, enabling context-aware code suggestions, explanations, and transformations without leaving the editor or switching to external tools.

## Key Aspects

- Direct integration with VS Code and JetBrains IDEs eliminates copy-paste overhead between AI tools and code editors
- Context-aware suggestions operate on the entire codebase rather than isolated code snippets, improving relevance
- Platforms like AICodeMirror provide multi-node disaster recovery and stable access to Claude models for IDE-based workflows
- Pricing models include monthly subscriptions and pay-per-use, with enterprise procurement support (invoices, contracts)
- Claude model tiers (lightweight/fast, hard-working, powerful) allow selecting the right balance of speed and performance per task
- Reduces friction in the development loop: read, edit, and confirm code changes entirely within the IDE

# Python NumPy Tutorial (Jupyter/Colab)

## Definition
The Python NumPy Tutorial is Justin Johnson's widely-referenced hands-on tutorial for learning NumPy, the fundamental numerical computing library in Python. It teaches array operations, broadcasting, indexing, and linear algebra through executable Jupyter/Colab notebooks, serving as a foundational resource for anyone entering scientific computing or machine learning.

## Key Aspects
- NumPy provides the ndarray, a high-performance multi-dimensional array object that is the backbone of scientific computing in Python
- Covers array creation, indexing, slicing, reshaping, and broadcasting rules
- Demonstrates vectorized operations that avoid explicit Python loops for massive performance gains
- Includes linear algebra operations: dot products, matrix multiplication, eigenvalue decomposition, SVD
- Jupyter/Colab notebooks allow interactive execution, making concepts immediately testable
- Broadcasting rules define how arrays of different shapes interact in arithmetic operations
- Essential prerequisite for pandas, scikit-learn, TensorFlow, PyTorch, and virtually all Python ML libraries
- The tutorial emphasizes idiomatic NumPy: thinking in arrays rather than loops

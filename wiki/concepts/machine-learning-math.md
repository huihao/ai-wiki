# Machine Learning Math

## Definition

Machine learning math is the collection of mathematical foundations required to understand, implement, and advance machine learning algorithms: linear algebra (vectors, matrices, transformations), calculus (gradients, chain rule, optimization), probability and statistics (distributions, Bayesian reasoning, hypothesis testing). These foundations enable practitioners to read research papers, implement models from scratch, and diagnose training failures -- without graduate-level mathematics.

## Key Aspects

- **Four-Stage Learning Path**: Foundational Math (algebra, calculus, linear algebra, probability) -> Classical ML -> Deep Learning -> Cutting-Edge ML; foundational math is the "make-or-break moment"
- **Linear Algebra**: Vectors, matrices, matrix multiplication, eigenvalues, least squares -- core to neural network operations (embeddings are matrix lookups, attention is matrix multiplication)
- **Calculus**: Derivatives, partial derivatives, chain rule, gradients -- essential for understanding backpropagation and gradient descent
- **Probability and Statistics**: Probability distributions, conditional probability, Bayes' theorem -- foundations of how LLMs model language as probability distributions
- **Classical ML Provides Intuition**: Loss functions, gradient descent, model architecture patterns make equations in modern papers immediately recognizable
- **Applied Linear Algebra**: Boyd & Vandenberghe's free textbook (Stanford ENGR108/UCLA EE133A) focuses on vectors, matrices, and least squares with practical applications
- **No Graduate Math Needed**: Undergraduate-level math suffices for implementing new models from papers; advanced math only needed for theoretical ML research
- **Implement from Scratch**: Key learning comes from coding streamlined versions of each model, not just watching lectures; deliberate practice builds real understanding

# AI Tool Directory

## Definition
An AI tool directory is a curated aggregation platform that catalogs, categorizes, and enables discovery of AI tools across domains such as conversational AI, image generation, video production, writing, and programming. These directories reduce search friction for users seeking appropriate AI solutions from an increasingly fragmented ecosystem.

## Key Aspects
- Large-scale directories aggregate thousands of tools (5,000+ on aishenqi.net, 27,000+ on Creati.ai)
- Categories span conversational AI, AI painting/design, video production, writing/office automation, and coding
- Include not just tools but MCP servers and autonomous agent packages, reflecting the shift toward agent-based architectures
- Serve both experts and beginners with curated resource discovery and latest AI news
- Function as meta-platforms: they don't provide the AI capabilities themselves but organize access to them

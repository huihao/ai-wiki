# GAN Checkerboard Artifacts

## Definition

Checkerboard artifacts are a type of visual artifact that appears in images generated by GANs, caused by the use of deconvolution (transposed convolution) layers for upsampling. The artifacts manifest as a regular, grid-like pattern overlaid on generated images, arising from uneven overlap when kernel size is not divisible by stride.

## Key Aspects

- **Root Cause**: Transposed convolution layers produce uneven coverage of input pixels when kernel size is not evenly divisible by stride, creating periodic overlap patterns
- **Visual Appearance**: A regular grid-like pattern across generated images, most visible in smooth or low-frequency regions
- **Affected Layers**: Primarily affects upsampling layers in the generator network
- **Mitigation Strategies**: Use resize-then-convolve (bilinear upsampling followed by regular convolution) instead of transposed convolution; or carefully tune kernel size and stride to ensure even overlap
- **Relevance Beyond GANs**: The phenomenon highlights a general principle about transposed convolution behavior in any generative architecture

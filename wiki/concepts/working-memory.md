# Working Memory

## Definition

The system responsible for holding and manipulating information for immediate cognitive tasks. In SOAR architecture, working memory stores declarative knowledge — facts and information representing the system's current understanding of its environment. In modern LLM agents, the context window serves as a functional analog of working memory.

## Key Aspects

- Holds immediate, task-relevant information
- Limited capacity (analogous to context window limits in LLMs)
- Active manipulation of stored information
- Distinct from long-term memory (persistent storage)

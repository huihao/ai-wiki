# Binary Analysis
## Definition
The process of examining compiled executable files (binaries) to understand their behavior without access to source code. Binary analysis involves disassembly, decompilation, and dynamic analysis to understand what a program does.
## Key Concepts
- **Static analysis**: Examining binary code without execution
- **Dynamic analysis**: Running and observing program behavior
- **Disassembly**: Converting machine code to assembly mnemonics
- **Decompilation**: Attempting to reconstruct higher-level source code

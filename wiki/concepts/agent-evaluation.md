# Agent Evaluation

## Definition

Agent evaluation is the systematic assessment of AI agent performance, measuring how effectively agents accomplish tasks, handle edge cases, and maintain reliability across diverse scenarios. It encompasses both automated benchmarks and human judgment of agent outputs.

## Key Aspects

- Evaluation requires clearly defined success criteria aligned with task requirements
- Benchmarks should test not just final outputs but intermediate reasoning and tool usage quality
- Specialist agents need domain-specific evaluation metrics beyond general task completion
- Human-in-the-loop evaluation remains essential for subjective quality assessment
- LLM-as-a-Judge patterns can scale evaluation but introduce their own biases
- Evaluation should cover reliability (consistency), efficiency (resource usage), and safety (harm avoidance)

# On-Device AI

AI models deployed directly on edge devices (mobile phones, tablets, embedded systems) without cloud dependency.

## Characteristics

- **Offline**: No internet required
- **Private**: Data never leaves device
- **Fast**: No network latency
- **Low power**: Optimized for battery efficiency

## Applications

### Mobile

- Virtual assistants
- Real-time translation
- Code completion
- Document summarization
- Healthcare symptom checking

### Edge Computing

- IoT smart home devices
- Industrial automation
- Autonomous vehicles
- Sensor data processing

## Advantages

- Enhanced privacy and security
- Reduced latency
- Offline operation capability
- Lower cloud costs
- Data sovereignty compliance

## Limitations

- Model size constraints
- Limited compute resources
- Battery/power consumption
- Narrower scope vs. cloud models
- Harder to update/upgrade

## Deployment Platforms

- **PocketPal**: iOS/Android app
- **Ollama**: Desktop PC
- **llama.cpp**: Multi-platform
- **MLC LLM**: Mobile-optimized

## Model Types

- SLMs (1-10B parameters)
- Quantized models (INT4, INT8)
- Distilled models
- Task-specific fine-tuned models

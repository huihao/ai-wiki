# Interactive Articles

## Definition

Interactive articles are a format of technical communication that combines written explanations with embedded interactive visualizations, code experiments, and manipulable diagrams to create deeply engaging educational content. Pioneered by publications like Distill, interactive articles allow readers to explore concepts by adjusting parameters, seeing real-time results, and engaging with living code examples rather than static text and images.

## Key Aspects

- **Distill Publication**: Active 2016-2021, pioneered the interactive article format for machine learning; featured articles on feature visualization, attention mechanisms, t-SNE, and GAN interpretability
- **Visualization-First Approach**: Emphasis on making abstract concepts concrete through interactive diagrams, not just illustrating text with static figures
- **Embedded Code**: Working code examples and live experiments embedded directly in articles, not just code blocks
- **Reader Manipulation**: Readers can adjust parameters, see how outputs change, and build intuition through exploration
- **Notable Works**: Feature visualization and activation atlases (Chris Olah), attention mechanism explanations, t-SNE visualization guide (Martin Wattenberg), graph neural networks explained
- **Technical Stack**: Articles built with D3.js (2011-2015), Vue.js (2015+), Canvas, KaTeX for math, and pointer events for interactivity
- **Impact on ML Communication**: Established a standard for how ML research should be communicated -- understanding over performance numbers
- **Legacy**: Despite the publication hiatus, the format has influenced how ML concepts are taught across the web

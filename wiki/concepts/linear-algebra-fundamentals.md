# Linear Algebra Fundamentals

## Overview

Mathematical foundation for machine learning, data science, and engineering applications. Vectors, matrices, and least squares provide core toolkit for understanding and implementing ML algorithms.

## Core Topics

### Vectors

- Definition: Ordered lists of numbers
- Operations: Addition, scalar multiplication, dot products
- Applications: Data representation, feature vectors, embeddings
- Properties: Norms, angles, orthogonality

### Matrices

- Definition: Rectangular arrays of numbers
- Operations: Multiplication, transposition, inversion
- Applications: Transformations, datasets, linear mappings
- Properties: Rank, determinant, eigenvalues, singular values

### Least Squares

- Definition: Optimization method minimizing sum of squared errors
- Applications: Regression, fitting, approximation
- Methods: Normal equations, QR decomposition, SVD
- Extensions: Weighted least squares, regularized variants

## Machine Learning Applications

**Data representation:**
- Vectors: Single data points, features
- Matrices: Datasets, batches, transformations

**Operations:**
- Linear transformations: Neural network layers
- Projections: Dimensionality reduction
- Decompositions: PCA, SVD, factorizations

**Optimization:**
- Least squares: Regression problems
- Gradient descent: Iterative optimization
- Regularization: L1/L2 penalties

## Specific ML Uses

**Supervised learning:**
- Linear regression: Least squares fitting
- Logistic regression: Linear transformations + activation
- Neural networks: Matrix multiplications per layer

**Unsupervised learning:**
- PCA: Eigendecomposition/SVD
- Clustering: Vector distances
- Embeddings: Vector representations

**Deep learning:**
- Weight matrices: Layer parameters
- Batch processing: Matrix operations
- Attention: Matrix multiplications (Q, K, V)

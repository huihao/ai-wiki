# Competitive Programming

Popular and challenging activity where participants compete to solve programming problems within time constraints.

## Format

Participants receive series of long problem descriptions and limited time (few hours) to write programs solving them.

**Problem types**:
- Placing roads and buildings within constraints
- Creating strategies to win custom board games
- Algorithm implementation
- Logic puzzles

**Ranking**: Based primarily on number of problems solved

## Participants

Hundreds of thousands participate worldwide to:
- Gain experience
- Showcase skills
- Collaborate in fun ways

## Industry Use

- Recruiting tool for companies
- Similar problems appear in software engineer hiring processes
- Skill benchmarking

## AI Achievement

[[AlphaCode]] first AI system to reach competitive level, placing at median competitor level on [[Codeforces]] contests (December 2022).

## Requirements

Problem-solving abilities require:
- Implementing algorithms
- Inventing algorithms (most difficult part)
- Critical thinking
- Logic
- Natural language understanding

# Contextual Suggestions

## Definition

Contextual suggestions refer to an AI system's ability to proactively generate relevant, situation-aware recommendations or actions based on a user's current context, history, and environment, rather than waiting for explicit requests. This shifts AI assistants from reactive response generators to proactive collaborators that anticipate user needs.

## Key Aspects

- Moves beyond reactive Q&A to proactive assistance that anticipates what the user needs next
- Leverages conversation history, user preferences, task context, and environmental signals to generate timely suggestions
- Requires understanding of user intent and workflow patterns to surface useful recommendations without overwhelming
- Balances helpfulness with intrusiveness -- suggesting too much or too little both degrade the user experience
- Relevant to AI coding assistants (suggesting next actions), knowledge systems (recommending related information), and general-purpose agents
- Often implemented using retrieval-augmented generation pipelines that combine contextual information with generative models

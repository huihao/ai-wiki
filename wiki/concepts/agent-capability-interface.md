# Agent Capability Interface (ACI)

## Definition

A design philosophy for AI Agent tools that focuses on making tool descriptions accurate, concise, and unambiguous so the model can correctly select and invoke them. ACI principles emphasize that most tool selection errors stem from inaccurate descriptions rather than model weakness — a tool's description is its "API contract" with the LLM.

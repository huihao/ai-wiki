# Cynefin Framework

## Definition

A sense-making framework that categorizes problems into five domains: Clear (obvious cause-effect), Complicated (expert analysis needed), Complex (understood only in retrospect), Chaotic (no perceivable order), and Confusion (uncertain which domain applies). Chris Richardson applied it to argue that GenAI coding agents operate in the Complex domain, requiring safe experimentation rather than standardized best practices.

# LLM Leaderboards

Platforms ranking model performance across multiple benchmarks and evaluation dimensions. Enable comparison of frontier AI models through various methodologies including automated benchmarks and crowdsourced human preference data.

## Key Platform: [[lmarena|LMArena]]

- **Methodology**: Crowdsourced benchmarks from real user interactions
- **Features**: Side-by-side model comparison, chat with multiple LLMs simultaneously
- **Scale**: Tracks over 300 models from frontier to legacy
- **Data source**: Real-world user preference rather than synthetic test sets

# Spaced Repetition

## Definition

A learning technique that incorporates increasing intervals of time between subsequent review of previously learned material in order to exploit the psychological spacing effect. The method is typically implemented through flashcard software (e.g., Anki) that schedules reviews based on the learner's performance.

## Key Proponents / Critics

- **Researchers**: Hermann Ebbinghaus (originator of the forgetting curve), Piotr Woźniak (SuperMemo algorithm), Andy Matuschak, Michael Nielsen
- **Practitioners**: [[radek-osmulski]] — applied the method to machine learning education via aiquizzes
- **Tools**: [[anki]] — most widely used open-source spaced repetition software

## Related Concepts

- Forgetting curve — the empirical basis for spaced repetition
- Active recall — the complementary technique of retrieving information from memory
- [[machine-learning|Machine learning education]] — domain where spaced repetition has been applied
- [[anki]] — software implementation

## Evolution

- **1885**: Hermann Ebbinghaus first describes the forgetting curve
- **1985**: Piotr Woźniak develops the SuperMemo algorithm
- **2006**: Anki is released as open-source spaced repetition software
- **2010s–2020s**: Widespread adoption in medical education, language learning, and technical fields
- **2020s**: Applied to machine learning concept mastery (aiquizzes)

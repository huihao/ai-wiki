# Plugin Marketplace

## Definition
A plugin marketplace is a centralized platform where developers can discover, install, and manage extensions (plugins or MCP servers) that augment the capabilities of an AI coding agent or development tool. Marketplaces provide a curated ecosystem of integrations covering databases, cloud services, browser automation, search, security, and communication tools, typically installable with a single click.

## Key Aspects
- Provides a broad ecosystem of integrations across categories: databases, cloud services, development tools, browser automation, search, communication, media, and security
- Enables AI coding agents (such as Cline) to interact with external systems through standardized protocols like MCP (Model Context Protocol)
- Notable marketplace categories include database/storage connectors (Supabase, PostgreSQL, Redis), cloud services (AWS suite, Cloudflare), browser tools (Playwright, Puppeteer), search engines (Perplexity, Brave), and communication platforms (Slack, Discord)
- Supports single-click installation and configuration through the host tool interface
- Key MCP servers include Context7 (documentation), Desktop Commander (file/terminal operations), Sequential Thinking (structured problem-solving), and Knowledge Graph Memory (persistent entity storage)

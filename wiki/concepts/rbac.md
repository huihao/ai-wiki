# RBAC

## Definition

Role-Based Access Control — a method of regulating access to resources based on the roles of individual users. In enterprise AI coding agent deployments, RBAC separates admin and user access levels, controlling who can configure Codex settings, manage team access, and approve code review workflows.

## Key Aspects

- Access controlled by role, not individual identity
- Admin roles: workspace configuration, team management, approval workflows
- User roles: task creation, code review participation
- Separation of duties for enterprise governance

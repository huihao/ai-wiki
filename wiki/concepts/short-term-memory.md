# Short-Term Memory (STM)

## Definition

Memory holding current context and immediate information needed for ongoing agent operations. Limited in size but directly accessible in agent's working context.

## Characteristics

- Holds current context
- Immediate accessibility
- Limited capacity
- Directly in agent's working memory
- Fast retrieval

## Integration with LTM

In unified memory systems like AgeMem, STM is combined with long-term memory inside the agent, allowing seamless transitions between immediate context and stored history.

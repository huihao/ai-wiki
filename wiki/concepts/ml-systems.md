# ML Systems

ML Systems refers to the engineering discipline of building, deploying, and operating machine learning models in production. Unlike ML research focused on algorithms and architectures, ML systems encompasses the full stack: data pipelines, feature engineering, model training infrastructure, hardware-aware optimization, inference serving, monitoring, and lifecycle management. The field addresses the gap between training a model in a notebook and running it reliably at scale for millions of users.

## Key Components
- Data engineering: collection, cleaning, versioning, and pipeline orchestration
- Model training: distributed training, hyperparameter optimization, experiment tracking
- Inference: serving, batching, quantization, hardware acceleration
- Monitoring: drift detection, performance metrics, incident response
- MLOps: CI/CD for models, reproducibility, governance

# LLM-Visualized

Interactive 3D and 2D visualization tool for understanding GPT-2 (124M) architecture and operation.

## Overview

LLM-Visualized provides an interactive exploration of how GPT-2 processes tokens through its architecture. The visualization shows:

- Token processing flow
- Layer-by-layer transformations
- Attention mechanism visualization
- KV cache behavior
- Generation process animation

## Features

### Interactive Exploration

Users can:
- Navigate through different processing stages
- Adjust token positions and generation settings
- Visualize attention patterns
- Explore layer internals

### Visualization Components

- **3D architecture view** - Spatial representation of model structure
- **2D flow diagrams** - Processing pipeline visualization
- **Token state tracking** - How embeddings evolve through layers
- **Attention matrix visualization** - Token-to-token attention weights

### Educational Value

The tool helps understand:
- How transformers process sequential data
- Attention mechanism mechanics
- Information flow through layers
- Context building during generation
- KV cache role in inference

## Technical Details

- Model: GPT-2 (124M parameters)
- Visualization: Real-time 3D/2D rendering
- Parameters: Token position, generation step, KV cache visibility
- Interface: Web-based interactive visualization

## Usage

Access the visualization at: https://www.llm-visualized.com/

Query parameters allow customization:
- `token` - Token position to visualize
- `generation` - Generation step number
- `kvCache` - KV cache visualization toggle

# nbdev - Create delightful software with Jupyter Notebooks

> Original URL: [https://nbdev.fast.ai/](https://nbdev.fast.ai/)

---

**Development framework by fast.ai** - Write prose, code, and tests in notebooks without context-switching.

## Core Features

- **Beautiful technical documentation** with Quarto
- **Out-of-the-box CI** with GitHub Actions
- **Publish to PyPI and conda** (code) and GitHub Pages (prose)
- **Two-way sync** with favorite IDEs
- **Git-friendly notebooks**: human-readable merge conflicts, no unwanted metadata

## Philosophy

Single environment for writing prose, code, and tests — no need to switch between notebooks, IDEs, and documentation tools.

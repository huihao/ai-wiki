# Mathematics for Machine Learning

> Source: [Mathematics for Machine Learning](https://mml-book.github.io/)

**Authors:** [[Marc Peter Deisenroth]], [[a-aldo-faisal|A. Aldo Faisal]], [[Cheng Soon Ong]]

**Publisher:** Cambridge University Press (April 2020)

**Official Site:** [mml-book.com](https://mml-book.com)

## Summary

A comprehensive textbook covering the mathematical foundations necessary for understanding machine learning. The book motivates learning mathematical concepts and provides the skills needed to read advanced ML texts, rather than covering advanced techniques themselves.

**Key Principle:** PDFs remain freely available alongside the printed version.

## Structure

**Part I: Mathematical Foundations**
- Linear algebra
- Calculus
- Probability theory
- Optimization

**Part II: Central Machine Learning Problems**
- Application of mathematical foundations to ML algorithms
- Practical problem-solving approaches

## Philosophy

- **Motivated learning:** Not just "how" but "why" mathematics matters
- **Accessible:** Assumes basic mathematical maturity
- **Foundation-first:** Build skills to tackle advanced ML texts
- **Concise:** Kept deliberately short to focus on essentials

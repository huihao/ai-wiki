# Welcome to the UC Irvine Machine Learning Repository

## Metadata

- **Maintainer:** UC Irvine
- **Date:** 2009-10-07
- **URL:** https://archive.ics.uci.edu/
- **Access Date:** 2026-04-27
- **Raw File:** `raw/ai/Welcome to the UC Irvine Machine Learning Repository.md`

## Summary

The UC Irvine Machine Learning Repository is one of the oldest and most widely used collections of datasets for machine learning research. It currently maintains 689 datasets covering diverse domains including medicine, finance, agriculture, and computer vision.

## Key Takeaways

- One of the earliest known ML dataset repositories (established 1987, website 2009)
- Currently maintains 689 datasets
- Classic datasets: Iris (Fisher 1936), Heart Disease, Wine Quality, Bank Marketing, Breast Cancer Wisconsin, Adult (Census Income)
- Recent additions include: vision-language benchmark COREVQA, heart/lung sounds dataset, agricultural datasets
- New website in beta at archive-beta.ics.uci.edu
- Datasets used by millions of researchers worldwide

## Raw Notes

- Iris: small classic dataset, one of earliest for classification methods
- Heart Disease: combines 4 databases (Cleveland, Hungary, Switzerland, VA Long Beach)
- Wine Quality: physicochemical tests predicting wine quality
- Bank Marketing: Portuguese banking direct marketing campaigns
- COREVQA: 5608 image-statement pairs for visual entailment reasoning
- HLS-CMDS: 535 heart and lung sound recordings from clinical manikin
- Paddy Dataset: hybrid ML model for rice yield prediction

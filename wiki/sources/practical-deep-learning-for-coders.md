## Summary

A free, practical deep learning course designed for coders with some programming experience. Uses a "top-down" teaching approach: students start with complete, state-of-the-art working systems and progressively dig deeper into foundations. Covers computer vision, NLP, tabular data, and collaborative filtering using PyTorch and the fastai library.

## Key Takeaways

- **Teaching philosophy**: Top-down learning. Start with working code that solves real problems, then gradually unpack how it works. Opposite of the traditional "start with linear algebra" approach.
- **Prerequisites**: One year of coding experience (preferably Python), high school math. No university math or special hardware required.
- **Course structure**:
  - **Part 1** (9 lessons, ~90 min each): CV, NLP, tabular data, collaborative filtering, model deployment
  - **Part 2**: Deep Learning Foundations to Stable Diffusion (>30 hours, for more experienced students)
- **Tools taught**: PyTorch, fastai, Hugging Face Transformers, Gradio
- **Key techniques**: Transfer learning, SGD, data augmentation, weight decay, random forests, gradient boosting, embeddings
- **Deployment**: Models turned into web applications using Gradio and deployed on free tiers (Kaggle, Paperspace)
- **Impact**: Alumni at Google Brain, OpenAI, Adobe, Amazon, Tesla. Research published at NeurIPS. Multiple Kaggle gold medalists.
- **Book**: *Deep Learning for Coders with fastai and PyTorch* (O'Reilly, freely available online)

## Raw Notes

- The course is explicitly anti-credentialist. Jeremy Howard argues that deep learning should be accessible to anyone who can code, regardless of formal background.
- The "myths" table (lots of math, lots of data, expensive computers) directly challenges common barriers to entry.
- Part 2 goes much deeper into foundations, including building Stable Diffusion from scratch.
- fastai is positioned as a higher-level library on top of PyTorch, similar to how Keras related to TensorFlow.

# So You Want to Learn LLMs? Here's the Roadmap

## Summary

A practical, no-nonsense 5-phase roadmap for learning LLMs, designed for people with a CS background who want to skip generic ML prerequisites. The guide covers foundations (linear algebra, PyTorch), transformer architecture, scaling laws, alignment/PEFT, and inference optimization, with hands-on projects at each stage.

## Key Takeaways
- 5-phase learning path: Foundations, Transformers, Scaling, Alignment, Inference
- Learn by layering: build intuition, strengthen theory, do hands-on, read papers, build real things
- Key projects: build Micrograd, build a mini-GPT from scratch, implement LoRA from scratch
- Core resources: 3Blue1Brown, Karpathy's Micrograd/Let's Build GPT, Stanford CS224N, "Attention Is All You Need" paper
- RLHF and PEFT (LoRA/QLoRA) are essential for real-world LLM use
- Scaling laws (Kaplan et al, Chinchilla) explain how LLMs got good

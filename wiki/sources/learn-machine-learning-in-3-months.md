## Summary

A 2018 self-paced curriculum for learning machine learning in three months, originally promoted via YouTube. The curriculum mixes free online resources: MIT linear algebra and probability courses, algorithm design, Udacity's Intro to ML, Siraj Raval's own video playlists, and fast.ai's deep learning course.

## Key Takeaways

- **Structure**: 3-month self-study curriculum combining video lectures, MOOCs, and project work.
- **Prerequisites covered**: Linear algebra (MIT 18.06 OCW), probability (MIT 6.041x), algorithms (Penn).
- **ML introduction**: Udacity Intro to Machine Learning.
- **Deep learning**: Siraj Raval's YouTube playlists + fast.ai deep learning course.
- **Project component**: Re-implementing DL projects from Siraj's GitHub repositories.
- **Community**: References a "Wizards" Slack channel for learners.
- **Historical note**: This was one of the more popular free ML curricula in the 2017–2019 period, though the author later faced controversies regarding academic integrity.

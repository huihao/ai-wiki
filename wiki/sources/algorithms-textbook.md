## Summary

A free electronic textbook and lecture notes collection for theoretical computer science courses at the University of Illinois, Urbana-Champaign. This material serves as the primary reference for CS 374 (Introduction to Algorithms & Models of Computation) and CS 473 (Advanced Algorithms). The textbook assumes familiarity with discrete math (especially induction) and basic data structures and algorithms (especially recursion), consistent with prerequisites CS 173 and CS 225. Bug reports and feedback are managed through GitHub.

## Key Takeaways

- **Prerequisites**: Not intended as a first introduction—requires discrete math and basic data structures knowledge
- **Course alignment**: Primary reference for two theoretical CS courses at Illinois (CS 374, CS 473)
- **Coverage gaps**: Does not cover stacks, queues, dynamic arrays, priority queues, balanced search trees, hash tables, amortized analysis, or other fundamental data-structure topics (see preface for details)
- **Open access**: Anyone can download, print, use, copy, and/or distribute without permission; redistribution should include a link back to the web page or algorithms.wtf shortcut
- **Bug tracking**: Issues managed via GitHub repository (jeffgerickson/algorithms)
- **Historical archive**: Complete archive of past homeworks, exams, and lab handouts available on separate page

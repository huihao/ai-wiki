## Summary

Official documentation for LlamaIndex.TS — the TypeScript implementation of the LlamaIndex framework for building LLM applications with context engineering. Supports Node.js, Deno, Bun, and Cloudflare Workers.

## Key Takeaways

- LlamaIndex.TS provides the same core capabilities as the Python version: data connectors, indexes, retrievers, agents, workflows, and observability.
- Built for modern JavaScript runtimes with idiomatic TypeScript APIs.
- Agents are LLM-powered assistants that reason, use tools, and take actions.
- Workflows are multi-step, event-driven processes combining agents, connectors, and tools.
- Context engineering is the umbrella term for injecting the right data into LLM prompts — RAG is the most popular example.
- LlamaCloud offers managed parsing (LlamaParse), extraction (LlamaExtract), and indexing.

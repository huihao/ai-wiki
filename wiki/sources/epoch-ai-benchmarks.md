# Epoch AI Benchmarks

## Metadata

- **Author**: Epoch AI
- **Date**: 2026-01-01

- **Fetched**: 2026-04-27

## Summary

A comprehensive index aggregating 50+ AI benchmarks into a single, general capability scale. Covers mathematics, coding, agentic action, and more domains.

## Key Benchmarks

### Mathematics & Reasoning

- **FrontierMath**: 50 exceptionally difficult research-level math problems
- **MATH Level 5**: Hardest tier from MATH dataset (AMC 10, AMC 12, AIME)
- **OmniMATH**: 300 expert-written math problems (undergrad through early career research)
- **OTIS**: 45 competition-style math problems
- **GSM8K**: Grade-school math word problems for multi-step arithmetic reasoning

### Coding & Software Engineering

- **SWE-bench**: GitHub issues from real-world Python repos, testing code fix implementation
- **SWE-bench Verified**: 500 GitHub issues with verified fixes
- **Exercism**: Challenging programming problems from online education platform
- **BigCodeBench**: Nonstandard ML engineering tasks across domains
- **SWE-agent**: Community-run evaluation of end-to-end software agents
- **SWE-1**: Duration benchmark for longest task models can complete reliably
- **MLAgentBench**: Measuring how well CLI agents can post-train small base LLMs

### Agentic Capabilities

- **OSWorld**: Computer-use benchmark with real desktop/web tasks
- **WebArena**: Tasks using computer terminal, testing program understanding
- **OSWorld-Computer-Use**: Agents complete tasks using keyboard/mouse actions
- **CyberSecEval**: Autonomous vulnerability discovery and exploitation

### Knowledge & Reasoning

- **MMLU**: Multi-task exam covering dozens of academic/professional subjects
- **GPQA**: Expert-authored questions spanning 100+ academic subjects
- **MMLU-Pro**: Harder, bias-reduced multiple-choice benchmark
- **HellaSwag**: Adversarially filtered commonsense sentence completion
- **ARC-AGI**: Few-shot abstract reasoning and pattern generalization
- **ARC-AGI-2**: Harder successor to ARC-AGI with emphasis on compute efficiency
- **BBH (Big-Bench Hard)**: Curated subset of BIG-bench tasks testing compositional, symbolic, multi-step reasoning
- **PIQA**: Physical commonsense benchmark for feasible everyday solutions

### Multimodal

- **MMMU**: Comprehensive multimodal video question answering benchmark
- **ScienceWorld**: Multimodal multiple-choice science benchmark with text, images, diagrams
- **GeoGuessr**: Identify real-world location from images
- **Text-to-CAD**: Generate valid parametric 3D designs meeting geometric/rendering checks

### NLP & Language Understanding

- **BoolQ**: Yes/no reading comprehension with naturally occurring questions
- **WinoGrande**: Large-scale pronoun resolution and coreference
- **TriviaQA**: Open-domain QA with challenging trivia and evidence documents
- **Drop**: Long-context language modeling benchmark
- **QuALITY**: Long creative writing comprehension

### Specialized Domains

- **FinanceBench**: Economically valuable knowledge work (investment banking, consulting, law, medical care)
- **Chess Puzzles**: 100 novel programmatically generated chess puzzles
- **SimpleQA**: 1,000 factoid questions across politics, science, art, sports, geography, music
- **GPQA Diamond**: PhD-level expert-authored questions in biology, chemistry, physics
- **OpenBookQA**: Open-book science QA requiring core facts + commonsense

### Evaluation & Writing

- **LMSYS Chatbot Arena**: Human voting on anonymous model website generation
- **Writing Quality Benchmark**: Multi-genre composition scoring (clarity, coherence, style, instruction-following)
- **LiveBench**: Dynamic real-world tasks with periodic releases and standardized judging

## Key Themes

- **Capability Testing**: Covers math, coding, reasoning, knowledge, multimodal understanding
- **Agentic Evaluation**: Growing focus on autonomous task completion and tool use
- **Real-World Tasks**: Shift from synthetic benchmarks to realistic challenges
- **Dynamic Evaluation**: LiveBench and similar benchmarks update regularly
- **Multimodal Integration**: Testing across text, images, video, code, 3D

# llmware Hugging Face Collection

## Metadata

- **URL**: https://huggingface.co/llmware
- **Date**: 2023-10-23
- **Fetched**: 2026-04-27
- **Description**: Largest open source collection of OpenVINO models optimized for AI PC deployment

## Summary

LLMWare maintains the largest open-source collection of OpenVINO models optimized for edge deployment (AI PCs). Specializes in enterprise LLM applications, middleware, and specialized models.

## Model Collections

### Model Depot
Optimized deployment for edge platforms (OpenVINO, ONNX, GGUF). See: [Model Depot article](https://medium.com/@darrenoberst/model-depot-9e6625c5fc55)

### SLIMs
Small, specialized function-calling models designed for stacking in multi-model, agent-based workflows. See: [SLIMs article](https://medium.com/@darrenoberst/slims-small-specialized-models-function-calling-and-multi-model-agents-8c935b341398)

### BLING/DRAGON
Highly-accurate fact-based question-answering models optimized for enterprise use cases. Benchmarks available: [small model benchmark](https://medium.com/@darrenoberst/best-small-language-models-for-accuracy-and-enterprise-use-cases-benchmark-results-cf71964759c8) | [development journey](https://medium.com/@darrenoberst/building-the-most-accurate-small-language-models-our-journey-781474f64d88)

### Industry-BERT
Industry fine-tuned embedding models for domain-specific applications.

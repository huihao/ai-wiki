## Summary

A five-phase roadmap for learning LLMs with a CS background, skipping generic ML prerequisites. Covers building an autograd engine, a mini-GPT, distributed training, alignment/fine-tuning, and inference optimization.

## Key Takeaways

- Phase 0: Foundations — linear algebra, probability, Micrograd, PyTorch basics.
- Phase 1: Transformers — tokenization, embeddings, self-attention, build mini-GPT.
- Phase 2: Scaling — scaling laws, data/tensor/pipeline parallelism, multi-GPU training.
- Phase 3: Alignment — RLHF, Constitutional AI, LoRA/QLoRA, fine-tune on real data.
- Phase 4: Production — FlashAttention, quantization, sub-second inference.
- Recommended resources: 3Blue1Brown, Karpathy, Stanford CS224N, "Attention Is All You Need."

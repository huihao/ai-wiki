## Summary

Guide to five beginner-friendly ML projects using real-world datasets. Each project includes dataset recommendation, goal, skills to practice, and suggestions for building APIs and dashboards.

## Projects

1. **House Price Prediction** (Regression) — Ames Housing Dataset
2. **Tweet Sentiment Analysis** (NLP Classification) — Twitter Sentiment Dataset
3. **Customer Segmentation** (Clustering) — Online Retail Dataset
4. **Customer Churn Prediction** (Imbalanced Classification) — Telco Customer Churn
5. **Movie Recommendation** (Recommender Systems) — MovieLens Dataset

## Key Takeaways

- Real-world data is messy — projects teach cleaning, EDA, and feature engineering
- Going beyond notebooks to APIs (Flask/FastAPI) and dashboards (Streamlit/Plotly) builds end-to-end skills
- Each project targets a different ML problem type: regression, NLP, clustering, classification, recommendation

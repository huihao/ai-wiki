# HAL Science Paper (blocked by anti-bot protection)

## Summary

The HAL science paper could not be accessed because the website uses Anubis anti-bot protection with a proof-of-work mechanism (Hashcash-style) to prevent AI scraping. The page returned a challenge page instead of the actual paper content.

## Key Takeaways

- HAL science (hal.science) uses Anubis anti-bot protection to prevent large-scale content scraping by AI companies
- Anubis uses a Hashcash-style proof-of-work mechanism that is negligible for individual users but costly for large-scale crawlers
- The protection is described as a placeholder solution while more sophisticated fingerprinting methods (e.g., headless browser font rendering detection) are developed
- JavaScript must be enabled and browser extensions like JShelter may need to be disabled to pass the challenge

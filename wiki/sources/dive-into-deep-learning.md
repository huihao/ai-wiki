# Dive into Deep Learning

## Metadata

- **Authors**: Aston Zhang, Zachary C. Lipton, Mu Li, Alexander J. Smola
- **Date**: Published 2023 by Cambridge University Press
- **URL**: https://d2l.ai/
- **Fetched**: 2026-04-27

- **GitHub**: https://github.com/d2l-ai/d2l-en

## Summary

Dive into Deep Learning (D2L) is an interactive deep learning book combining code, mathematics, and discussions. It is implemented in PyTorch, NumPy/MXNet, JAX, and TensorFlow, and has been adopted at 500 universities across 70 countries. The book is forthcoming from Cambridge University Press and has become a best-seller in Chinese markets.

## Key Takeaways

- **Format**: Interactive book where each section is an executable Jupyter notebook
- **Implementations**: Available in PyTorch, NumPy/MXNet, JAX, and TensorFlow
- **Adoption**: Used at 500 universities from 70 countries
- **Publication**: Published by Cambridge University Press (2023)
- **Chinese version**: Best-seller at largest Chinese online bookstore
- **Content structure**:
  - Preliminaries and linear neural networks
  - Multilayer perceptrons and modern CNNs (AlexNet, VGG, ResNet, DenseNet)
  - Recurrent neural networks and modern RNNs (LSTM, GRU)
  - Attention mechanisms and Transformers
  - Computer vision applications
  - Natural language processing (word2vec, BERT)
  - Reinforcement learning, Gaussian processes, hyperparameter optimization
  - GANs and recommender systems

## Notable Features

- **Interactive learning**: Modify code and tune hyperparameters with instant feedback
- **Integrated approach**: Mathematics + Figures + Code + Text + Discussions
- **Community support**: Discussion forums linked in each section
- **Multi-framework**: Same concepts implemented across multiple frameworks
- **Course materials**: Berkeley course materials (slides, notebooks, assignments, videos) available

## Raw Notes

The book represents a shift in ML education from static textbooks to interactive, executable content. The multi-framework approach (PyTorch, JAX, TensorFlow) allows learners to choose their preferred tooling while learning the same concepts. The adoption at 500 universities demonstrates the effectiveness of the interactive approach.

The Chinese version's success indicates strong demand for accessible deep learning education in Chinese markets. The open-source GitHub repository enables continuous updates and community contributions.

# Transformers - Hugging Face Documentation

## Summary

The official documentation hub for the Hugging Face Transformers library, which serves as the model-definition framework for state-of-the-art ML models across text, vision, audio, video, and multimodal tasks. The library centralizes model definitions so they are compatible across the ecosystem: training frameworks (Axolotl, Unsloth, DeepSpeed, FSDP), inference engines (vLLM, SGLang, TGI), and adjacent libraries (llama.cpp, mlx).

## Key Takeaways
- Transformers is the pivot model-definition framework across the ML ecosystem
- Over 1M+ model checkpoints available on Hugging Face Hub
- Three main classes per model: configuration, model, and preprocessor
- Key features: Pipeline (optimized inference), Trainer (mixed precision, FlashAttention, distributed training), generate() (fast text generation with streaming and decoding strategies)
- Recommended learning path: LLM course covering fundamentals to practical applications
- Models are designed for both inference and training with minimal code

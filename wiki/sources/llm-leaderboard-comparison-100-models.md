## Key Metrics

### Intelligence Index

Composite score measuring model capability across various benchmarks and tasks.

### Output Speed

Tokens per second - throughput rate of generated content.

### Latency (TTFT)

Time To First Token - how quickly the model starts generating response.

### Price

Cost per 1M tokens (blended 3:1 input-to-output ratio).

### Context Window

Maximum number of tokens the model can process in a single request.

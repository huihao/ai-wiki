## Summary

Hermes Agent is a self-improving AI agent from Nous Research with a built-in learning loop. It creates skills from experience, improves them during use, nudges itself to persist knowledge, searches past conversations, and builds a deepening model of the user across sessions.

## Key Takeaways

- Built-in learning loop: autonomous skill creation after complex tasks, skills self-improve during use, FTS5 session search with LLM summarization.
- Multi-platform messaging: Telegram, Discord, Slack, WhatsApp, Signal, Email, plus CLI with full TUI.
- Model-agnostic: supports 200+ models via OpenRouter, plus NVIDIA NIM, Xiaomi MiMo, z.ai/GLM, Kimi/Moonshot, MiniMax, Hugging Face, OpenAI, and custom endpoints.
- Scheduled automations, delegates/parallelizes tasks, runs on $5 VPS to GPU clusters.
- OpenClaw migration support via `hermes claw migrate`.
- RL training integration via Atropos/Tinker environments.

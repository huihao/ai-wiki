## Summary

Website providing deep, visual explanations of machine learning topics. Includes open-source tools (TensorSensor, dtreeviz, rfpimp, StratPD) and articles on RNNs, regularization, gradient boosting, matrix calculus, and feature importance.

## Key Takeaways

- TensorSensor: Clarifies tensor shape mismatches by visualizing code execution across JAX, PyTorch, TensorFlow, NumPy
- dtreeviz: Scikit-learn decision tree visualization with feature space splits
- Matrix calculus article: Complete reference for differential matrix calculus in deep learning
- Philosophy: "illuminate not impress" — clarity over complexity
- StratPD/CatStratPD: Model-agnostic partial dependence curves computed directly from data

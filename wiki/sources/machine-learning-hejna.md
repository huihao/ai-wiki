> Source: [Machine Learning by Joey Hejna](https://joeyhejna.com/mlbook/intro.html)

**Author:** [[Joey Hejna]]

## Summary

A comprehensive set of notes on machine learning that assumes very little background knowledge in linear algebra or probability. The series takes readers from zero background knowledge through neural networks.

**Status:** Work in progress

## Key Features

- Assumes minimal prerequisites (linear algebra and probability basics)
- Progressive learning path: foundations → neural networks
- Comprehensive coverage designed for self-study

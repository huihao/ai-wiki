# chrome-read-later

## Summary

A Chrome browser extension for managing a temporary reading list, focused on saving pages to read later rather than closing and forgetting them. Features Vim-style keybindings, reading/video progress tracking, cloud sync via Google account, local history, and JSONL export/import. Designed for power users who want keyboard-driven tab management.

## Key Takeaways
- Temporary bookmark system as an alternative to closing tabs and losing them
- Vim keybindings: j/k navigation, o/O to open, dd to delete, u to undo, yy to copy link
- Tracks reading progress (scroll position) and video progress (playback time and rate)
- Cloud sync across Chrome browsers with Google account (up to 250 items)
- JSONL export/import for backup and transfer
- Right-click context menu to save links or current page
- Keyboard shortcuts: Alt+Shift+S to save, Alt+Shift+Z to zoom open/close

# Introduction to Machine Learning

Princeton University COS 324 course notes covering machine learning fundamentals, prepared by professors Sanjeev Arora, Danqi Chen and undergraduates Simon Park and Dennis Jacob.

## Course Structure

The course is organized into six major parts:

### Part 1: Supervised Learning

Fundamentals of supervised learning where models learn from labeled training data to make predictions on new data.

### Part 2: Unsupervised Learning

Methods for finding patterns and structure in unlabeled data, including clustering and dimensionality reduction.

### Part 3: Deep Learning

Neural network architectures, training techniques, and applications of deep learning methods.

### Part 4: Reinforcement Learning

Learning through interaction with an environment, where agents learn optimal behavior through rewards and penalties.

### Part 5: Advanced Topics

Advanced machine learning concepts and modern techniques in the field.

### Part 6: Mathematics for Machine Learning

Mathematical foundations essential for understanding machine learning:

**Chapter 18: Probability and Statistics**
- Probability theory fundamentals
- Statistical methods
- Statistical inference

**Chapter 19: Calculus**
- Differential calculus
- Integral calculus
- Optimization methods

**Chapter 20: Linear Algebra**
- Vector spaces
- Matrix operations
- Linear transformations

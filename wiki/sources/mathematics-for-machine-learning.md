# Mathematics for Machine Learning

## Metadata

- **Author**: Marc Peter Deisenroth, A. Aldo Faisal, Cheng Soon Ong
- **Date**: 2020-04
- **Publisher**: Cambridge University Press

## Summary

A book that motivates mathematical concepts by connecting them to machine learning applications. Not aimed at advanced ML techniques (those are covered elsewhere), but at providing the mathematical skills needed to read advanced ML literature. Freely available as PDF.

## Structure

- **Part I: Mathematical Foundations** — linear algebra, calculus, probability
- **Part II: Central Machine Learning Problems** — regression, classification, dimensionality reduction, density estimation

## Key Takeaways

- Designed to be concise — doesn't cover everything, focuses on what's essential
- Keeps PDFs freely available even after print publication
- Endorsed by prominent researchers: Joelle Pineau, Christopher Bishop, Pieter Abbeel

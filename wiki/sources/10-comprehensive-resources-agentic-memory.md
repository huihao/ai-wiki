# 10 Comprehensive Resources about Agentic Memory

## Summary

A curated list of 10 resources covering agentic memory: surveys, papers, and videos on memory types, LLM architectures, security risks, and benchmarks. Compiled to help AI engineers and researchers understand how memory works in AI agents.

## Key Takeaways

- Agentic memory is becoming a new retrieval layer for agents' knowledge, distinct from simple RAG
- Long-term memory in LLM agents has significant security risks: manipulation, leakage, corruption across systems
- Graph-based representations are useful for organizing hierarchical knowledge and improving retrieval in self-evolving agents
- Current agentic memory systems often fail to match their theoretical promise due to benchmark limitations and latency costs
- AI needs hippocampus-like memory (separate from model weights) for object permanence and on-device intelligence
- Storage, not compute, is becoming the key bottleneck for building dependable AI agents (Edo Liberty, Pinecone)
- MemEvolve framework jointly adapts agent experience and memory architecture for improved transfer across tasks

## Key Resources

1. **Rethinking Memory Mechanisms of Foundation Agents** (Stanford, Salesforce, Google, Meta) — comprehensive survey of memory types, architectures, evaluation methods
2. **Security of Long-Term Memory in LLM Agents** — threat taxonomy across the memory lifecycle (writing, storing, retrieving, sharing, deleting)
3. **Graph-based Agent Memory** — graphs for entity/relationship representation and hierarchical knowledge organization
4. **Anatomy of Agentic Memory** — practical evaluation of memory systems, benchmark limitations, latency costs
5. **The AI Hippocampus** — implicit vs explicit vs agentic persistent memory, multimodal settings
6. **Shawn Shen on Visual Memory** — embodied AI needs separate hippocampus-like memory
7. **Edo Liberty (Pinecone)** — storage as the key bottleneck for AI agent memory beyond RAG
8. **Does AI Remember?** — SOAR's legacy, memory types, generative workflows
9. **MemEvolve** — meta-evolution of agent memory systems with EvolveLab
10. **Memory in the Age of AI Agents** — taxonomy across memory form, function, and dynamics

# What is Amazon SageMaker AI?

## Metadata

## Summary

AWS documentation overview of Amazon SageMaker AI, a fully managed machine learning service. As of December 2024, Amazon SageMaker was rebranded as part of a broader "Amazon SageMaker" platform that unifies data, analytics, and AI capabilities. The AI-specific component retains the original ML model building, training, and deployment features.

## Key Takeaways

- SageMaker AI is a fully managed ML service for building, training, and deploying models
- Rebranded from "Amazon SageMaker" on December 3, 2024; legacy namespaces remain unchanged
- The broader "Amazon SageMaker" platform now includes: SageMaker AI, Lakehouse, Data/AI Governance, SQL Analytics, Data Processing, Unified Studio, and Bedrock
- Provides managed ML algorithms that run efficiently on distributed infrastructure
- Supports bring-your-own algorithms and frameworks
- Offers flexible distributed training options
- Models can be deployed into secure, scalable production environments

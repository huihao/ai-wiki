## Overview

Invariant Labs is a company focused on AI agent security. Their mission is to help agent builders create reliable, robust, and secure products. They were acquired by Snyk in June 2025 to accelerate agentic AI security innovation.

## Key Research Areas

### MCP Security

Invariant Labs has done extensive security research on the Model Context Protocol:

- Identified critical vulnerabilities
- Developed MCP-Scan for security scanning
- Demonstrated real-world exploits (GitHub, WhatsApp)
- Proposed tool poisoning attack methodology

### Agent Security Frameworks

- **Guardrails**: Contextual security layer
- **AgentDojo**: Benchmarking framework for utility + security
- **Explorer**: Observability and debugging tool
- **Invariant Gateway**: Security layer for production

### Formal Verification

Research on formal security guarantees for AI agents:
- Hard constraints on agent behavior
- Mathematical verification of security properties
- Integration with existing agent frameworks

### Browser Agent Safety

Specialized research on browser-based agents:
- Navigation safety
- Data exfiltration prevention
- UI interaction security

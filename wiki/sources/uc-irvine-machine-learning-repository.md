# UC Irvine Machine Learning Repository

## Metadata

## Summary

The UC Irvine Machine Learning Repository is a classic dataset repository serving the machine learning community since 1987. Maintains 689 datasets as a service to ML researchers and practitioners worldwide. Provides curated datasets for classification, regression, clustering, and other ML tasks.

## Key Takeaways

- Repository contains 689 datasets across multiple domains
- Classic datasets: Iris (1936), Heart Disease, Wine Quality, Bank Marketing, Breast Cancer Wisconsin, Adult
- One of earliest and most referenced dataset sources in ML literature
- Popular for benchmarking and educational purposes
- New datasets added regularly: Amazon/Google reviews, medical datasets (HLS-CMDS), vision benchmarks (COREVQA)
- Categories: biology, medicine, finance, social sciences, physical sciences
- New beta website available: archive-beta.ics.uci.edu

## Popular Datasets

- **Iris** (1936): Classic classification benchmark from Fisher
- **Heart Disease**: 4 databases (Cleveland, Hungary, Switzerland, VA Long Beach)
- **Wine Quality**: Red/white vinho verde samples from Portugal
- **Bank Marketing**: Portuguese banking direct marketing campaigns
- **Breast Cancer Wisconsin**: Diagnostic database
- **Adult/Census Income**: Income prediction from census data

## Recent Datasets

- Amazon Product and Google Locations Reviews (time series)
- HLS-CMDS: Heart and lung sounds from clinical manikin (535 recordings)
- COREVQA: Vision entailment benchmark (5608 image-statement pairs)
- Amaranthus Viridis leaves: CNN model evaluation dataset
- Paddy Dataset: Rice crop yield prediction
- PGCB Hourly Generation: Bangladesh electricity grid data

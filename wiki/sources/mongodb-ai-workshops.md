## Summary

Six-part hands-on workshop series covering modern AI application development with Java, Spring AI, and MongoDB Atlas. The series progresses from MongoDB fundamentals through vector search, retrieval-augmented generation (RAG), and AI agent orchestration.

## Key Takeaways

- **Workshop 1**: MongoDB document model fundamentals and distributed architecture.
- **Workshop 2**: CRUD operations in Java using MongoDB Query API (SQL-to-document translation).
- **Workshop 3**: Aggregation pipeline for complex data processing with Spring Data.
- **Workshop 4**: Vector Search with Spring AI — embedding generation and vector indexing in MongoDB Atlas.
- **Workshop 5**: RAG with MongoDB and Spring AI — retrieval workflows and Spring AI Advisors for LLM-powered apps.
- **Workshop 6**: AI Agents with MongoDB and Spring AI — orchestration, agent memory, and real-time database integration.
- **Stack**: Java + Spring AI + MongoDB Atlas.
- **Format**: Live sessions with Q&A; recordings available on-demand.

# Windsurf

## Summary

Windsurf is an AI-powered integrated development environment (IDE) built on a VS Code foundation. It provides AI-driven code generation, editing, and reasoning capabilities designed to assist developers with complex coding tasks. Windsurf competes with Cursor and other AI-native editors by offering an agentic coding experience.

## Key Attributes
- **Type**: AI-powered IDE
- **Notable for**: agentic coding workflow with AI-driven code generation and multi-file editing
- **Other facts**: built on VS Code; formerly known as Codeium; offers both free and paid tiers; positions itself as an alternative to Cursor for AI-assisted development

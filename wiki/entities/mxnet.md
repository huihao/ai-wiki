# MXNet

## Summary

Apache MXNet is a deep learning framework supporting both imperative and symbolic programming, known for its efficiency and scalability, used as one of the backend frameworks in the Dive into Deep Learning (D2L) textbook.

## Key Attributes

- Apache Software Foundation project
- Supports both imperative (PyTorch-like) and symbolic (TensorFlow-like) programming
- Efficient multi-GPU and distributed training
- Used as one of D2L textbook's three framework backends (alongside PyTorch and JAX)
- Created by contributors including Mu Li

# D3.js

## Summary

D3.js (Data-Driven Documents) is a JavaScript library for producing dynamic, interactive data visualizations in web browsers using HTML, SVG, and CSS, widely used for scientific and educational visualizations.

## Key Attributes

- Created by Mike Bostock, Vadim Ogievetsky, and Jeffrey Heer
- Binds data to DOM elements for data-driven visualization
- Supports SVG, Canvas, and HTML rendering
- Used in Red Blob Games for interactive math/CS visualizations

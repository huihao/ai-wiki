# Weights and Biases

## Metadata

- **Type**: ML Platform/Tool
- **Purpose**: Experiment tracking and model management
- **URL**: https://wandb.ai

## Description

Weights and Biases (W&B) is a machine learning platform for experiment tracking, dataset versioning, and model management. It provides visualization tools, collaboration features, and integration with various ML frameworks.

## Key Features

- Experiment tracking with metrics visualization.
- Hyperparameter sweeps and optimization.
- Dataset versioning and artifact management.
- Model registry and deployment tracking.
- Team collaboration and sharing.
- Integration with popular ML frameworks (PyTorch, TensorFlow, etc.).

## Use Cases

- Tracking RL training progress (used in Vibe RL experiments).
- Monitoring reward curves and error metrics.
- Comparing different training configurations.
- Analyzing checkpoint performance.

# ChatGPT

## Summary

OpenAI's conversational AI assistant based on GPT models, launched in November 2022. It popularized consumer-facing LLM interaction and became the fastest-growing application in history, reaching 100 million users within two months.

## Key Attributes

- **Developer**: OpenAI
- **Base Models**: GPT-3.5, GPT-4, GPT-4o (and their variants)
- **Interface**: Web chat, mobile apps, API
- **Key Features**: Conversational AI, code generation, image analysis (GPT-4V), plugin ecosystem, custom GPTs, memory, web browsing, DALL-E integration
- **Impact**: Catalyzed mainstream adoption of LLMs; sparked global conversation about AI safety, regulation, and the future of work

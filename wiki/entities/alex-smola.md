# Alexander Smola

## Summary

Alexander J. Smola is a machine learning researcher and professor known for co-authoring "Dive into Deep Learning" (D2L), an interactive, open-source deep learning textbook adopted at 500 universities across 70 countries. He has held positions at Carnegie Mellon University, Amazon Web Services, and other leading institutions, contributing extensively to kernel methods, statistical learning theory, and scalable ML systems.

## Key Attributes

- **Role**: Researcher, professor, textbook author
- **Affiliation**: Carnegie Mellon University (formerly), Amazon Web Services (formerly)
- **Notable work**: Co-authored "Dive into Deep Learning" (D2L) with Aston Zhang, Mu Li, and Zachary Lipton
- **Research areas**: Kernel methods, statistical learning, deep learning, scalable machine learning
- **D2L role**: One of four co-authors of the interactive deep learning textbook

# Hugging Face

## Summary

Hugging Face is an open-source AI platform and community best known for the Transformers library, the Hub for sharing models and datasets, and Spaces for hosting interactive ML demos. It has become the central repository for open-weight language, vision, and multi-modal models.

## Key Attributes

- **Founded**: 2016
- **Headquarters**: New York, USA / Paris, France
- **Key products**: Transformers, Datasets, Tokenizers, Accelerate, PEFT, Diffusers, Hub, Spaces
- **Notable models**: BERT, GPT-2, Llama, Mistral, Falcon hosted on the Hub
- **Community**: 1M+ models, 200K+ datasets, 500K+ demo Spaces

# Continue

## Summary

Continue is an open-source VSCode extension that integrates AI coding assistants into the IDE. It supports multiple model providers including Ollama for local LLMs, OpenAI, and others.

## Key Attributes

- **License**: Open source
- **Platform**: VSCode
- **Key features**: Code autocomplete, chat, documentation generation, refactoring
- **Model support**: Ollama, OpenAI, Anthropic, and more

- Ollama — local model provider
- VSCode — host IDE

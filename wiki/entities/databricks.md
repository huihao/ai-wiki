# Databricks

## Summary
Databricks is a unified data and AI platform built by the original creators of Apache Spark. It provides a lakehouse architecture that combines the capabilities of data lakes and data warehouses, along with integrated tools for machine learning, data engineering, and analytics. Databricks acquired MosaicML in 2023, gaining expertise in training and fine-tuning large language models, and has since developed DBRX, an open-source LLM optimized for enterprise use.

## Key Attributes
- **Type**: data and AI platform company
- **Notable for**: Lakehouse platform built on Apache Spark; DBRX open-source LLM; MLflow originator
- **Founded**: 2013
- **Headquarters**: San Francisco, California, USA
- **Key products**: Databricks Lakehouse, MLflow, DBRX, Unity Catalog

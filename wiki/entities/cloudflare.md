# Cloudflare

## Summary
Cloudflare is a global cloud services provider originally known for CDN, DNS, and DDoS protection, now expanded into serverless computing (Workers), storage (R2), and AI inference. Cloudflare Workers AI provides a serverless platform for running AI models at the edge with low latency. Cloudflare's Workers platform and its global network of 300+ data centers make it a strong option for deploying AI-powered applications close to end users.

## Key Attributes
- **Type**: cloud infrastructure and security company
- **Notable for**: edge AI inference via Workers AI; serverless edge computing
- **Founded**: 2009
- **Headquarters**: San Francisco, California, USA
- **Key products**: CDN, Workers, Workers AI, R2 storage, DDoS protection

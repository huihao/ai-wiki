# Agent Patterns

## Summary

Agent patterns are recurring design patterns and architectural approaches used in building AI agent systems. These patterns address common challenges such as tool use, planning, reflection, multi-agent coordination, and error recovery. They provide a vocabulary and shared understanding for designing effective agentic systems.

## Key Attributes
- **Type**: design patterns / architectural concepts
- **Notable for**: providing reusable solutions to common agent design challenges
- **Other facts**: common patterns include ReAct (reasoning + acting), reflection, tool use, planning loops, hierarchical delegation, and human-in-the-loop; patterns can be combined to build complex agent workflows

# V8 Engine
## Summary
Google's open-source JavaScript and WebAssembly engine used in Chrome and Node.js. V8 compiles JavaScript directly to native machine code using a 4-tier JIT compilation pipeline: Ignition (interpreter), Sparkplug (baseline), Maglev (mid-tier), and TurboFan (optimizing compiler).
## Key Attributes
- **Type**: JavaScript/WebAssembly engine
- **Developer**: Google
- **Used in**: Chrome, Node.js, Deno
- **Key features**: Hidden classes, inline caching, speculative optimization, deoptimization

# Tony Alicea

## Summary

Tony Alicea is a developer and educator who created Trace, a declarative modeling standard for AI code generation. Trace organizes software specifications into a standardized folder structure designed to produce precise AI-generated code while teaching systems thinking to developers.

## Key Attributes

- **Known for**: Creator of Trace (declarative spec standard)
- **Website**: tonyalicea.dev
- **Other work**: Author of "Don't Imitate, Understand" (free online book on UI normalization/denormalization)

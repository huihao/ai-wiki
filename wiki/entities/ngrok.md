# ngrok

## Summary
ngrok is a secure ingress platform for developers that provides tunnels, load balancing, and API gateway services. It enables developers to expose local servers to the internet through secure tunnels, and publishes a well-regarded technical blog covering infrastructure, APIs, LLM internals, and developer tools.

## Key Attributes
- **Type**: Developer tool / infrastructure platform
- **Notable for**: Secure tunnel service for exposing local development servers; excellent technical blog content
- **Founded**: 2015 (originally a side project by Alan Shreve)
- **Key features**: HTTPS tunneling, static domains, IP restrictions, OAuth, rate limiting, webhook inspection
- **Blog topics**: LLM prompt caching, quantization, API design, infrastructure patterns

# GitHub

## Summary

GitHub is the world's largest platform for software development and version control, built on Git. Acquired by Microsoft in 2018, GitHub has become a central hub for AI coding agents through GitHub Copilot, GitHub Actions CI/CD, and its vast open-source repository corpus. It is the primary distribution channel for open-source AI tools and a critical data source for training coding models.

- **Founded**: 2008

- **Infrastructure**: GitHub Actions (CI/CD), GitHub Codespaces, GitHub Models
- **Strategic importance**: Hosts most open-source ML/AI code; training data for Codex and Copilot

# Python

## Summary

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum and first released in 1991. Its design philosophy emphasizes code readability and simplicity, using significant whitespace indentation. Python has become the dominant language in data science, machine learning, artificial intelligence, and web development, supported by a vast ecosystem of libraries and frameworks.

## Key Attributes

- **Type**: Programming language
- **Created by**: Guido van Rossum (first released 1991)
- **Notable for**: Readability, extensive standard library, massive ecosystem; the lingua franca of AI/ML
- **License**: Python Software Foundation License (open-source)
- **Other facts**: Supports multiple paradigms (procedural, object-oriented, functional); key libraries include NumPy, pandas, PyTorch, TensorFlow, Django, and FastAPI; governed by a steering council since PEP 13

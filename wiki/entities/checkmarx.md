# Checkmarx

## Summary

Checkmarx is a security company specializing in application security, particularly in static application security testing (SAST) and software composition analysis (SCA). The company partners with Equixly to bridge the gap between traditional security testing and runtime security validation, addressing limitations in static-only security approaches.

## Key Attributes

- **Focus**: Application security, SAST, SCA
- **Partnership**: Collaborates with Equixly for continuous penetration testing
- **Limitation**: SAST/SCA insufficient for runtime security gaps (per Equixly analysis)
- **Market**: DevSecOps and application security testing

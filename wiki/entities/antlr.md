# ANTLR

## Summary

ANTLR (ANother Tool for Language Recognition) is a powerful parser generator for reading, processing, executing, and translating structured text or binary files, widely used for building language parsers, interpreters, and compilers.

## Key Attributes

- Open-source parser generator by Terence Parr
- Generates parsers for recognized grammars (BNF format)
- Supports Java, C#, Python, JavaScript, and other target languages
- Used in compilers, IDEs, and data processing pipelines

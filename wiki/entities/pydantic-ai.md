# Pydantic-AI

**Type**: Framework
**Category**: AI Agent Development, Framework
**Organization**: Pydantic Team

## Description

Framework for building AI agents with strong typing, validation, and utilities. Provides agent framework and helpful tools for immediate utility.

## Key Features

- **Agent Framework**: Core agent structure
- **Strong Typing**: Pydantic validation throughout
- **Helpful Utilities**: Immediate agent enhancements
- **MCP Support**: Model Context Protocol integration
- **CLI Interface**: Built-in CLI capabilities

## Core Components

- Agent class for agent structure
- MCP server integration
- Bedrock model support
- Tool definition utilities
- CLI interface generation

## Integration Capabilities

- AWS Bedrock (Claude models)
- Other mainstream providers
- Fully local LLM support
- MCP server connections
- Custom tool implementations

## Default MCP Server

- Sandboxed Python execution
- Safe code execution environment
- Run via Deno runtime
- Standard MCP interface

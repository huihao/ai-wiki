# Django

## Summary
Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. It follows the "batteries included" philosophy, providing built-in ORM, authentication, admin interface, URL routing, and templating. Django is used by major websites including Instagram, Pinterest, and Disqus, and remains one of the most widely adopted Python web frameworks for building data-driven web applications.

## Key Attributes
- **Type**: web framework (software)

- **Language**: Python
- **License**: BSD
- **First release**: 2005
- **Created by**: Adrian Holovaty and Simon Willison (originally for the Lawrence Journal-World newspaper)

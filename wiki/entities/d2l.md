# D2L (Dive into Deep Learning)

## Metadata

- **Type**: Book / Educational Resource
- **Authors**: Aston Zhang, Zachary C. Lipton, Mu Li, Alexander J. Smola
- **Publisher**: Cambridge University Press (2023)
- **URL**: https://d2l.ai/
- **GitHub**: https://github.com/d2l-ai/d2l-en

## Description

Dive into Deep Learning (D2L) is an interactive, open-source deep learning book adopted at 500 universities across 70 countries. Each section is an executable Jupyter notebook, allowing readers to modify code, tune hyperparameters, and receive instant feedback. The book is implemented in PyTorch, NumPy/MXNet, JAX, and TensorFlow.

## Key Features

- Interactive Jupyter notebook format (executable sections)
- Multi-framework implementations (PyTorch, JAX, TensorFlow, MXNet)
- Comprehensive coverage from fundamentals to advanced topics
- Mathematics + Figures + Code integrated approach
- Community discussion forums for each section
- Open-source with continuous updates via GitHub
- Published by Cambridge University Press (2023)
- Chinese version is best-seller in Chinese markets

## Adoption

- 500 universities across 70 countries
- Berkeley STAT 157 course materials available
- University list includes: MIT, Stanford, CMU, Berkeley, Harvard, and hundreds more worldwide

## Coverage

- Linear networks and perceptrons
- Modern CNNs (AlexNet, VGG, ResNet, DenseNet)
- RNNs and modern variants (LSTM, GRU)
- Attention mechanisms and Transformers
- Computer vision applications
- NLP (word2vec, GloVe, BERT)
- Reinforcement learning
- Gaussian processes
- Hyperparameter optimization
- GANs
- Recommender systems

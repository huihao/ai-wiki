# Invariant Labs

Invariant Labs is an AI security company dedicated to helping agent builders create reliable, robust, and secure products. They specialize in research and tooling for AI agent security, particularly around the Model Context Protocol (MCP).

## History

- **Founded**: 2024
- **Acquired by Snyk**: June 24, 2025
- **Mission**: Accelerate agentic AI security innovation

## Key Products

### Guardrails
Contextual security layer for the agentic era. Provides state-of-the-art security measures for AI agents.

### MCP-Scan
Security scanner designed to protect agentic applications using the Model Context Protocol. Identifies vulnerabilities and security risks in MCP implementations.

### Invariant Gateway
Seamless debugging and security layer for agentic applications. Provides observability and security monitoring.

### Explorer
Open-source observability tool to analyze agent behavior. Helps visualize and understand AI agent execution traces.

### AgentDojo
Framework for jointly evaluating security and utility of AI agents. Won first prize ($50,000) in the Center for AI Safety SafeBench competition.

## Research Contributions

### MCP Security Research

Invariant Labs discovered critical vulnerabilities in the Model Context Protocol:

1. **Tool Poisoning Attacks** (April 2025)
   - Critical vulnerability in MCP
   - New attack methodology

2. **GitHub MCP Exploit** (May 2025)
   - Demonstrated private repository access via MCP
   - Critical vulnerability in official GitHub MCP server

3. **WhatsApp MCP Exploit** (April 2025)
   - Showed message history exfiltration risks
   - Risks of untrusted MCP servers

### Agent Security Research

- Formal security guarantees for AI agents
- Hard constraints on agent behavior
- Browser agent safety measures
- Web agent trace analysis

## Acquisition by Snyk

On June 24, 2025, Snyk acquired Invariant Labs to:
- Deepen Snyk Labs research bench
- Integrate agent security into developer security tools
- Accelerate agentic AI security innovation

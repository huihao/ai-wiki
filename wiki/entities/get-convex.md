# Get Convex

## Summary

Get Convex is an AI agent skill provider listed in the Agent Skills Directory, offering installable skill packages that extend AI agent capabilities. With approximately 126,900 installations, it represents one of many third-party providers in the growing agent skills ecosystem alongside Microsoft, Firebase, and Lark Suite.

## Key Aspects

- **Agent Skill Provider**: Offers specialized skill packages for AI agents, installable through the skills.sh directory
- **Growing Ecosystem**: Part of a broader agent skills marketplace where providers package domain-specific capabilities into installable modules
- **Installation Metrics**: Approximately 126,900 installs, demonstrating demand for agent-extensible capabilities
- **Modular Architecture**: Skills are self-contained packages that add capabilities without modifying the core agent framework
- **Community Distribution**: Distributed through a centralized directory with installation tracking and leaderboards

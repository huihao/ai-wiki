# Pydantic

## Summary

Pydantic is a Python data validation library using type annotations, widely adopted in the LLM/AI ecosystem for defining tool schemas, validating API inputs, and ensuring type safety in agent frameworks.

## Key Attributes

- Python library using type hints for data validation
- Core dependency of FastAPI web framework
- Used to define tool/function schemas for LLM function calling
- Powers Structured Outputs in OpenAI and other LLM APIs
- Adopted by LangChain, CrewAI, and most AI agent frameworks
- Supports complex nested models, validators, and serialization

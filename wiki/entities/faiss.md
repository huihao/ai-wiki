# FAISS

## Summary

FAISS (Facebook AI Similarity Search) is a library by Meta AI for efficient similarity search and clustering of dense vectors, widely used for nearest neighbor search in recommendation systems and RAG pipelines.

## Key Attributes

- Created by Meta AI (Facebook AI Research)
- Optimized for dense vector similarity search
- Supports GPU acceleration for large-scale search
- Used in RAG pipelines for embedding retrieval
- Handles billions of vectors efficiently

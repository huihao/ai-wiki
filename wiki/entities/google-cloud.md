# Google Cloud

## Summary

Google Cloud Platform (GCP) is Google's suite of cloud computing services, providing infrastructure, AI/ML tools, and managed services for building and deploying applications at scale.

## Key Attributes

- Offers Vertex AI for model serving and deployment
- Provides GPU/TPU compute instances for ML training and inference
- Competes with AWS and Azure in cloud infrastructure market
- Referenced for GPU cloud computing, ML platforms, and model serving

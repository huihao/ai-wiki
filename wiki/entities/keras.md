# Keras
## Summary
A high-level deep learning API that runs on top of TensorFlow (and historically Theano/CNTK). Keras provides a user-friendly interface for building and training neural networks, making it popular for educational purposes and rapid prototyping.
## Key Attributes
- **Type**: Deep learning API
- **Backend**: TensorFlow (primary), JAX, PyTorch
- **Focus**: User-friendly, modular, extensible

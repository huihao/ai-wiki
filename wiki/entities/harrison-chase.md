# Harrison Chase

## Summary
Harrison Chase is the CEO and co-founder of LangChain, a widely adopted open-source framework for building applications powered by large language models. LangChain provides tools for chaining together LLM calls, managing prompts, integrating retrieval-augmented generation, and building agentic workflows, making it one of the most prominent frameworks in the LLM application development ecosystem.

## Key Attributes
- **Type**: person
- **Notable for**: Founding LangChain and building it into a leading LLM application framework
- **Company**: LangChain (also behind LangSmith and LangGraph)
- **Focus areas**: LLM application development, agent orchestration, retrieval-augmented generation

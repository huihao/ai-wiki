# SciPy

## Summary

SciPy (Scientific Python) is an open-source library of algorithms and mathematical tools for Python, built on top of NumPy. It provides modules for optimization, linear algebra, integration, interpolation, special functions, FFT, signal and image processing, ODE solvers, and statistics. SciPy is a core component of the Python scientific computing stack and is used extensively in academic research, engineering, and data science.

## Key Attributes

- **Type**: Python library (scientific computing)
- **Created by**: Travis Oliphant, Pearu Peterson, and Eric Jones (2001); maintained by SciPy community and NumFOCUS
- **Notable for**: Comprehensive mathematical algorithms; foundation for many higher-level ML and data science tools
- **License**: BSD-3-Clause
- **Other facts**: Key submodules include scipy.optimize, scipy.linalg, scipy.signal, scipy.stats, scipy.spatial, and scipy.sparse; required dependency for scikit-learn; commonly taught in university scientific computing courses

# UC Irvine

## Summary

UC Irvine (University of California, Irvine) is a major research university known for hosting the UC Irvine Machine Learning Repository, one of the oldest and most influential dataset repositories in machine learning history. The repository has served the ML community since 1987 with 689 curated datasets.

## Key Attributes

- **Type**: Public research university
- **Location**: Irvine, California
- **Notable Service**: Machine Learning Repository (archive.ics.uci.edu)
- **Repository Founded**: 1987
- **Dataset Count**: 689 datasets
- **Impact**: foundational resource for ML research and education

## Contributions

- Machine Learning Repository: classic benchmark datasets (Iris, Heart Disease, Wine Quality)
- Continuous dataset additions across domains
- Service to global ML community

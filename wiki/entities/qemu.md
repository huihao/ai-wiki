# QEMU

## Summary

QEMU (Quick Emulator) is a free and open-source machine emulator and virtualizer that can emulate full operating systems, used in Kubernetes education and development for creating virtual machine-based deployments.

## Key Attributes

- Open-source machine emulator and virtualizer
- Supports full system emulation and user-mode emulation
- Used in Kubernetes workshops for VM-based deployment scenarios
- Supports x86, ARM, RISC-V, and other architectures
- Integrates with KVM for hardware-accelerated virtualization

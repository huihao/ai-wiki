# Taipy

## Summary

Taipy is a Python framework for building data and business intelligence web applications. It enables production-ready applications with optimized performance, large data support, and multi-user capabilities.

## Key Attributes

- **Type**: Python framework
- **Domain**: Data visualization, Business Intelligence
- **Features**: Built-in decimator, caching control, multi-user support, VS Code extension
- **Enterprise**: Taipy Enterprise offering for security and scale

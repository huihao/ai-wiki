# Prime Intellect

## Summary

Prime Intellect provides compute and infrastructure for training, evaluating, and deploying agentic models. It offers hosted RL training, benchmark evaluations, serverless inference, and a GPU compute marketplace.

## Key Attributes

- **Products**: Hosted RL training, Evaluations, Deploy (inference), Environments Hub, Verifiers library
- **Compute**: On-demand 1-256 GPUs, reserved clusters 8-5000+ GPUs
- **Target**: Agentic model developers

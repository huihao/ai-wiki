# 13point5

## Metadata

- **Type**: Person/Handle
- **Context**: ML researcher/experimenter
- **Known for**: Vibe RL experiments, ASCII diagram alignment RL tasks

## Description

13point5 (Sriraam) is an ML experimenter who documented using Prime Intellect and OpenAI Codex to perform semi-autonomous reinforcement learning ("Vibe RL") on ASCII diagram alignment tasks, achieving ~2x performance improvement over GPT 4.1 mini with Qwen3-4B.

## Key Contributions

- Created synthetic dataset (7,200 samples) for ASCII diagram generation.
- Developed multi-component reward system (format_reward, alignment_reward, layout_spread_reward).
- Built RL Studio UI for inspecting rollouts and reward hacking.
- Open-sourced environment: https://github.com/13point5/ascii-align-env
- Pushed environment to Prime Intellect Hub: https://app.primeintellect.ai/dashboard/environments/13point5/ascii-align

# Second Mate

## Summary

Second Mate is a chatbot interface for LLMs designed to work with Obsidian knowledge bases, enabling conversational interaction with personal markdown wiki content.

## Key Attributes

- Chat interface for LLMs integrated with Obsidian
- Enables conversational querying of markdown knowledge bases
- Uses personal notes/documents as context for LLM responses
- Referenced in LLM knowledge base architecture discussions

# RunPod

## Summary

RunPod is a GPU cloud computing platform offering on-demand Pods, serverless inference, and reserved clusters at prices marketed as up to 80% cheaper than hyperscalers.

## Key Attributes

- **Products**: GPU Pods, Serverless, Clusters, Network Storage
- **Pricing model**: Per-second billing for on-demand; reserved commitments
- **Target**: ML training, inference, and AI workloads

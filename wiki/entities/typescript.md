# TypeScript

## Summary

TypeScript is an open-source programming language developed and maintained by Microsoft. It is a strict syntactical superset of JavaScript, adding optional static typing and class-based object-oriented programming to the language. TypeScript compiles to plain JavaScript and is widely used in large-scale frontend and backend development.

## Key Attributes
- **Type**: programming language (compiled to JavaScript)
- **Notable for**: adding static type checking to JavaScript, enabling safer refactoring and better tooling support
- **Other facts**: first released in 2012; designed by Anders Hejlsberg; the type system uses structural typing; widely adopted in the Node.js ecosystem for server-side development

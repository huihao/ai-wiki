# Princeton University

Princeton University is a leading research university offering comprehensive AI and machine learning education through courses like COS 324.

## AI/ML Education

**COS 324 — Introduction to Machine Learning**
Course taught by professors Sanjeev Arora and Danqi Chen covering:
- Supervised Learning
- Unsupervised Learning
- Deep Learning
- Reinforcement Learning
- Advanced Topics
- Mathematics for Machine Learning (Probability, Calculus, Linear Algebra)

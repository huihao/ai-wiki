# QoderWork

## Summary

An AI coding tool developed by Alibaba that supports tool use, browser connectivity, and integration with evaluation/test platforms. Used in the [[automated-evaluation-platform|automated evaluation platform]] described by [[feng-ling|凤聆]], where it drives the full evaluation lifecycle: reading platform skill descriptions, creating evaluation tasks, generating test suites, running evaluations, and submitting reports — all from a single natural-language instruction.

## Key Attributes

- **Organization**: Alibaba
- **Capabilities**: AI coding, browser connectivity (connects to user's browser with shared login state), integration with external platforms via skill descriptions
- **Use case**: Driving automated evaluation workflows end-to-end from natural language

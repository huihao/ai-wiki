# Metasploit
## Summary
A widely-used open-source penetration testing framework developed by Rapid7. Provides tools for exploit development, payload generation (including shellcode via msfvenom), and security auditing. The msfvenom tool is commonly used to generate shellcode for various platforms.
## Key Attributes
- **Type**: Penetration testing framework
- **Developer**: Rapid7
- **Key tool**: msfvenom for shellcode generation
- **Platform**: Cross-platform

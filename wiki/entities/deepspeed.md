# DeepSpeed

## Summary

DeepSpeed is a deep learning optimization library by Microsoft that provides system-level optimizations for training and inference, including ZeRO memory optimization, 3D parallelism, and efficient fine-tuning.

## Key Attributes

- Created by Microsoft Research
- ZeRO optimizer: reduces memory footprint for large model training
- 3D parallelism: combines data, model, and pipeline parallelism
- Supports mixed precision training and gradient compression
- Used with Hugging Face Transformers for large model training

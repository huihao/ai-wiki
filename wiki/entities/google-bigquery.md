# Google BigQuery

## Summary

Google BigQuery is a serverless, highly scalable multi-cloud data warehouse designed for large-scale data analytics, supporting SQL queries over petabytes of data with built-in ML capabilities (BigQuery ML).

## Key Attributes

- Serverless data warehouse (no infrastructure management)
- Supports petabyte-scale SQL analytics
- BigQuery ML for in-database model training
- Standard SQL with extensions
- Integration with Google Cloud ecosystem (Vertex AI, Dataflow, Dataproc)
- Referenced for join algorithm implementations at scale

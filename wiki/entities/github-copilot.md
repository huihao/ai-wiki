# GitHub Copilot

## Summary
GitHub Copilot is an AI-powered code completion and pair programming tool developed by GitHub in collaboration with OpenAI. Initially powered by the Codex model, Copilot provides real-time code suggestions, multi-file edits, and conversational coding assistance directly within supported IDEs. With Copilot Chat and Copilot Workspace, it has expanded from simple autocomplete into a more comprehensive AI coding assistant capable of generating entire functions, writing tests, debugging code, and explaining existing code.

## Key Attributes
- **Type**: AI coding assistant (software)
- **Notable for**: most widely adopted AI pair programming tool; integrated into VS Code, JetBrains, and other IDEs
- **Launched**: June 2021 (technical preview); general availability June 2022
- **Developer**: GitHub (Microsoft)
- **Underlying model**: OpenAI Codex / GPT-4 family

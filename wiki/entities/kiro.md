# Kiro

## Summary

Kiro is a lightweight spec-driven development tool by AWS that provides a Requirements → Design → Tasks workflow for AI-assisted software development, emphasizing structured specifications before code generation.

## Key Attributes

- Lightweight spec-driven development tool
- Provides Requirements → Design → Tasks workflow
- Created by Amazon Web Services (AWS)
- Emphasizes specifications before code generation
- Compared with spec-kit and Tessl for SDD approaches

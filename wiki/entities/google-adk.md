# Google ADK

## Summary
Google Agent Development Kit (ADK) is an open-source framework for building, testing, and deploying AI agents within the Google ecosystem. Released in 2025, ADK provides tools for creating agents powered by Gemini models, supporting multi-agent orchestration, tool integration, and deployment on Google Cloud. It is designed to work with the Model Context Protocol (MCP) and Google's broader AI infrastructure including Vertex AI.

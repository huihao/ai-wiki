# AWS Inferentia

## Summary

AWS Inferentia is Amazon's custom machine learning inference chip designed to deliver high-performance, low-cost ML inference at scale, used with TorchServe for serving PyTorch models.

## Key Attributes

- Custom ML inference ASIC by Amazon/AWS
- Designed for high throughput, low latency inference
- Used for Llama 2 deployments via TorchServe
- Offers cost advantages over GPU-based inference for large-scale workloads

# Pocket Flow

**Type**: Framework
**Category**: LLM Framework, Agent Development
**Organization**: The-Pocket

## Description

Minimalist 100-line LLM framework designed for building agentic applications. Provides simple abstractions for creating flow-based AI systems.

## Core Abstractions

1. **Nodes**: Individual units of computation performing specific tasks
2. **Flows**: Directed graphs of nodes defining execution paths
3. **Shared Store**: Dictionary accessible to all nodes for data sharing

## Key Features

- Minimalist design (100 lines of core code)
- Flow-based architecture for complex workflows
- Easy extensibility with custom nodes
- Clear separation of concerns
- YAML-based structured outputs

## Use Cases

- AI coding agents
- Decision-making systems
- Multi-step workflows
- Code analysis and modification
- File system operations

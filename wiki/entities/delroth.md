# delroth (Baptiste Daroussin)

## Summary

Software developer and reverse engineer known for his work on FreeBSD and LLVM, and for reverse engineering the script interpreter in "Tales of Symphonia: Dawn of the New World" (Wii). Created the cscript-interpreter project, a complete reimplementation of the game's custom bytecode VM.

## Key Attributes

- FreeBSD and LLVM contributor
- Reverse engineered Wii game script interpreter over 3-4 months
- Created cscript-interpreter GitHub project
- Expertise in dynamic analysis using Dolphin emulator

# nginx

## Summary
nginx (pronounced "engine-x") is a high-performance open-source web server, reverse proxy, load balancer, and HTTP cache. Originally created to address the C10K concurrency problem, nginx has become one of the most widely deployed web servers globally, serving as infrastructure for millions of websites and APIs.

## Key Attributes
- **Type**: Web server / reverse proxy / load balancer
- **Created**: 2004 by Igor Sysoev
- **Notable for**: Event-driven, non-blocking architecture enabling high concurrency with low memory usage
- **License**: BSD (open source); nginx Plus is the commercial version
- **Market share**: One of the top 2-3 web servers globally (alongside Apache)
- **Key features**: Reverse proxy, load balancing, SSL/TLS termination, HTTP/2 support, caching, WebSocket proxying
- **Company**: nginx, Inc. (acquired by F5 Networks in 2019)

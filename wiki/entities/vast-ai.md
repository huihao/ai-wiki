# Vast.ai

## Summary

Vast.ai is a GPU cloud computing platform providing affordable GPU rental services for AI and machine learning workloads. Operates a marketplace model connecting GPU owners with users needing compute resources for training, inference, and other compute-intensive tasks.

## Key Attributes

- **Type**: GPU cloud computing platform
- **Model**: Marketplace for GPU rental
- **Focus**: Affordable GPU access for AI/ML
- **Platform**: Web console for instance management
- **Use Cases**: Model training, inference, deep learning, scientific computing
- **Integration**: Stripe payments, Google authentication

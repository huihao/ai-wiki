# ONNX Runtime

## Metadata

- **Type**: ML Framework/Tool
- **Creator**: Microsoft
- **Purpose**: Cross-platform ML inference engine
- **URL**: https://onnxruntime.ai

## Description

ONNX Runtime is a high-performance inference engine for machine learning models in ONNX (Open Neural Network Exchange) format. It provides cross-platform support across multiple operating systems, APIs, and hardware accelerators for deploying trained ML models.

## Key Features

- Multi-platform support (desktop, mobile, web).
- Multiple API bindings (Python, C++, C, Java, JavaScript, etc.).
- Hardware acceleration (CPU, GPU, TPU, custom accelerators).
- ONNX format compatibility.
- Open source with active GitHub community.

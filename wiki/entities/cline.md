# Cline

**Type**: AI Coding Tool
**Category**: Development Tools, AI Agents
**Organization**: Open Source Project

## Description

Open-source AI coding agent with Plan/Act modes, MCP integration, and terminal-first workflows. Trusted by 5M+ developers worldwide for uncompromised coding assistance.

## Key Features

- **Open Source**: Full transparency and customization
- **MCP Marketplace**: Extensive plugin ecosystem
- **Plan/Act Modes**: Flexible operation styles
- **Terminal-First**: Developer-centric interface
- **Mass Adoption**: 5M+ developers worldwide

## Operation Modes

### Plan Mode
- Analyzes requests before action
- Shows intended changes
- Allows review and approval
- Safer for complex changes

### Act Mode
- Immediate execution
- Faster for simple tasks
- Direct code modification
- Efficient workflows

# Apache Kafka
## Summary
A distributed event streaming platform originally developed at LinkedIn and later open-sourced under the Apache Software Foundation. Kafka implements a publish-subscribe messaging system with producers, brokers, and consumers, designed for high-throughput, fault-tolerant, real-time data feeds.
## Key Attributes
- **Type**: Distributed event streaming platform
- **Origin**: LinkedIn
- **Architecture**: Producers, brokers, consumers
- **Key paper**: "Kafka: a Distributed Messaging System for Log Processing"

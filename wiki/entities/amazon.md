# Amazon (AWS SageMaker)

## Summary

Amazon Web Services (AWS) provides Amazon SageMaker AI, a fully managed machine learning platform for building, training, and deploying ML models. AWS is a leading cloud provider offering comprehensive ML infrastructure, tools, and services including SageMaker AI, Bedrock, and data analytics platforms.

## Key Attributes

- **Type**: Cloud computing provider
- **Key ML Service**: Amazon SageMaker AI (fully managed ML)
- **Parent**: Amazon.com, Inc.
- **ML Platform Evolution**: Unified SageMaker platform (Dec 2024)
- **Services**: SageMaker AI, Bedrock, Redshift, S3, EMR, Glue, Athena
- **Focus**: Enterprise ML deployment, infrastructure management

## ML Products

- **SageMaker AI**: Build, train, deploy ML models
- **Bedrock**: Foundation model service for generative AI
- **SageMaker Lakehouse**: Unified data access
- **SageMaker Unified Studio**: Integrated development environment
- **Training infrastructure**: Distributed GPU clusters
- **Inference**: Real-time and batch prediction endpoints

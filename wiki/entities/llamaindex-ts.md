# LlamaIndex.TS

TypeScript/JavaScript implementation of the LlamaIndex framework for building LLM-powered applications.

## Summary

LlamaIndex.TS is the official TypeScript port of the LlamaIndex data framework. It provides tools for connecting LLMs with external data through indexing, retrieval, and query engines. Built for Node.js, Deno, Bun, and Cloudflare Workers, it enables developers to build RAG applications, agents, and workflows in the JavaScript/TypeScript ecosystem.

## Key Features

- **Data connectors**: Ingest from APIs, files, SQL, and dozens more sources
- **Indexes & retrievers**: Store and retrieve data for LLM consumption
- **Agents and Engines**: Chat and reasoning interfaces over your data
- **Workflows**: Fine-grained orchestration of data and LLM-powered agents
- **Observability**: Built-in integrations for monitoring and debugging

## Use Cases

- LLM-powered agents with tool use
- RAG chatbots with indexed data
- Structured data extraction from documents
- Multi-step workflow orchestration

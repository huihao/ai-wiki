# Claude

## Summary

Claude is a family of large language models developed by Anthropic. The Claude model family includes Claude Opus, Claude Sonnet, and Claude Haiku, targeting different tiers of capability and speed. Claude is designed with a focus on AI safety and helpfulness, and powers products such as Claude.ai (chatbot), Claude Code (CLI coding agent), and the Anthropic API.

## Key Attributes
- **Type**: large language model family / AI assistant
- **Notable for**: safety-focused design, constitutional AI training methodology, and leading performance on coding, reasoning, and analysis benchmarks
- **Other facts**: developed by Anthropic; Claude 3.5 Sonnet and Claude 4 Sonnet have achieved strong benchmark results; Claude Code is a CLI-based agentic coding tool; Claude supports tool use, extended thinking, and multimodal input

# Matteo Collina

## Summary

Matteo Collina is a developer known for his work in the Node.js ecosystem and for creating pi-self-learning, a Pi extension that gives coding agents git-backed memory. The extension automatically extracts learnings from completed tasks and commits them to a dedicated memory repository.

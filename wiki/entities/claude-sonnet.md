# Claude Sonnet

## Summary

Anthropic's balanced model, offering a middle ground between the deep reasoning of Opus and the speed of Haiku. The most versatile option for general-purpose tasks, providing strong performance at moderate cost and latency.

## Key Attributes

- **Developer**: Anthropic
- **Positioning**: Balanced tier in the Claude family (between Opus and Haiku)
- **Strengths**: Strong capability across coding, analysis, and conversation at moderate cost
- **Default Model**: Often used as the primary model in Claude Code workflows
- **Use Cases**: Code generation, content creation, data analysis, chatbots, summarization, general Q&A
- **API Access**: Available through Anthropic's Messages API

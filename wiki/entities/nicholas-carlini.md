# Nicholas Carlini

## Summary

Nicholas Carlini is a researcher known for creating Yet Another Applied LLM Benchmark, a collection of nearly 100 tests extracted from actual conversation history with LLMs. The benchmark demonstrates that personal benchmarks based on real-world problems are viable and valuable.

## Key Attributes

- **Known for**: Yet Another Applied LLM Benchmark
- **GitHub**: github.com/carlini/yet-another-applied-llm-benchmark
- **Innovation**: DSL for specifying question asking and answer evaluation

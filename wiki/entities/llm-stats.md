# LLM Stats

> Platform tracking large language model releases, benchmarks, and ecosystem developments.

## Overview

LLM Stats provides daily AI model release news, benchmark updates, and research developments. Tracks over 500 models across commercial APIs and open source releases from major AI labs including OpenAI, Anthropic, Google, Meta, and Mistral.

## Features

- **Model release tracking** — Daily updates on new LLM releases
- **Benchmark coverage** — GPQA, HumanEval, MMLU evaluations
- **Provider monitoring** — OpenAI, Anthropic, Google, Meta, Mistral, open-weight labs
- **Ecosystem overview** — 500+ models catalogued
- **Research news** — Latest developments from leading labs

## Coverage Areas

1. **Commercial APIs**
   - OpenAI GPT-4 series
   - Anthropic Claude models
   - Google Gemini family

2. **Open Source/Open-Weight**
   - Meta Llama family
   - Mistral models
   - Leading research labs

3. **Benchmarks**
   - GPQA — Graduate-level reasoning
   - HumanEval — Code generation
   - MMLU — Multitask understanding

## Value

Helps developers track AI model updates, compare capabilities through benchmarks, and make informed model selection decisions. Real-world performance depends on specific use cases beyond benchmark metrics.

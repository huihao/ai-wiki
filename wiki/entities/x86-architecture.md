# x86 Architecture
## Summary
The x86 instruction set architecture (ISA) developed by Intel, widely used in desktop and server processors. The 32-bit variant uses general-purpose registers (EAX, EBX, ECX, EDX, ESI, EDI, ESP, EBP) and supports both Intel and AT&T assembly syntaxes.
## Key Attributes
- **Type**: CPU instruction set architecture
- **Developer**: Intel (originally), later AMD for x86-64 extension
- **Variants**: x86 (32-bit), x86-64/AMD64 (64-bit)
- **Register model**: General-purpose registers with sub-register access (AX, AH, AL)

# Data Commons

Data Commons aggregates and harmonizes global, open data, giving everyone the power to uncover insights with natural language questions.

## Overview

Data Commons is an initiative from Google that provides:
- Unified Knowledge Graph of global data
- Natural language query interface
- Statistical variable exploration
- Data visualization tools
- API access for programmatic use

### Exploration Tools

**Knowledge Graph** — Explore what data is available and understand the graph structure

**Statistical Variable Explorer** — Explore statistical variable details including metadata and observations

**Data Download Tool** — Download data for selected statistical variables

**Map Explorer** — Study how statistical variables vary across geographic regions

**Scatter Plot Explorer** — Visualize correlation between two statistical variables

**Timelines Explorer** — See trends over time for selected statistical variables

### Natural Language Queries

Users can ask natural language questions like:
- "Which counties in the US have the most smoke pollution?"
- "Which countries have the most greenhouse gas emissions?"
- "Tell me about the economy in Brazil."
- "What is the age distribution in the USA?"
- "How does life expectancy vary across countries in Africa?"

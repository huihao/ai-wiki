# Amazon Web Services (AWS)

## Summary

Amazon Web Services (AWS) is the world's largest cloud computing platform and a major provider of AI infrastructure and services. AWS offers Amazon Bedrock (managed foundation model API), Amazon SageMaker (ML platform), Trainium/Inferentia chips, and vast GPU clusters. It is the default cloud for many enterprise AI deployments and a key partner for companies building generative AI platforms.

## Key Attributes

- **Parent company**: Amazon
- **Launched**: 2006
- **Market position**: Largest cloud provider by revenue
- **Key AI services**: Amazon Bedrock, Amazon SageMaker, AWS Trainium, AWS Inferentia, Amazon Q
- **Key partnerships**: Anthropic ($4B investment), NVIDIA
- **Notable customers**: Formula 1 (generative AI RCA), Netflix, Airbnb

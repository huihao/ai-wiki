# pandas

## Summary

pandas is an open-source Python library providing high-performance data structures and data analysis tools. Built on top of NumPy, it offers the DataFrame and Series data structures for manipulating structured (tabular, time-series, heterogeneous) and unlabeled data. It is one of the foundational libraries in the Python data science ecosystem, widely used for data cleaning, transformation, aggregation, and exploratory data analysis.

## Key Attributes

- **Type**: Python library (open-source)
- **Created by**: Wes McKinney, first released in 2008
- **Notable for**: Intuitive DataFrame API that made Python a serious competitor to R and Excel for data manipulation
- **License**: BSD-3-Clause
- **Other facts**: Name derived from "panel data"; core dependency of the PyData ecosystem alongside NumPy and matplotlib

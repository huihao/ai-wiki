# Node.js Test Runner

## Metadata

- **Type**: Testing Framework
- **Platform**: Node.js
- **Built-in**: Native Node.js feature
- **Docs**: https://nodejs.org/api/test.html

## Definition

Node.js Test Runner is the built-in testing framework included in Node.js core, providing native test execution without requiring external dependencies like Jest or Mocha.

## Key Features

- Built into Node.js core.
- No external dependencies required.
- Simple API for test definition.
- Support for assertions.
- Async test handling.
- Test reporting and output.

## Benefits

- Zero installation overhead.
- Consistent with Node.js patterns.
- Minimal configuration.
- Works with modern Node.js versions.

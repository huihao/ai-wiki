# VS Code

## Summary

Visual Studio Code (VS Code) is a free, open-source code editor developed by Microsoft. It is built on the Electron framework and provides a rich ecosystem of extensions for programming languages, debugging, and version control. VS Code has become the most widely used code editor among developers, serving as the foundation for many AI coding assistants and custom configurations.

## Key Attributes
- **Type**: source code editor / integrated development environment
- **Notable for**: extensive extension marketplace, built-in Git integration, integrated terminal, and broad language support
- **Other facts**: first released in 2015; built on GitHub's Electron framework; available on Windows, macOS, and Linux; powers numerous AI coding extensions including Cursor and Continue

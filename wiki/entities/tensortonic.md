# TensorTonic

## Summary

TensorTonic is an ML learning platform offering over 200 algorithms to implement from scratch. It provides interactive visualizations, browser-based execution, and gamification elements to make learning machine learning algorithms engaging and hands-on. The platform also covers implementations of influential research papers, making it a comprehensive resource for understanding ML from first principles.

## Key Attributes

- **Type**: Educational platform (machine learning)
- **Notable for**: 200+ algorithms to implement from scratch; interactive visualizations; browser-based code execution; gamified learning experience
- **Other facts**: Covers classic ML algorithms, deep learning building blocks, and research paper implementations; designed for hands-on learners who want to understand the mechanics behind ML algorithms rather than just using libraries

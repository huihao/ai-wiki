# scikit-learn

## Summary

scikit-learn is an open-source Python machine learning library built on NumPy, SciPy, and matplotlib. It provides simple and efficient tools for predictive data analysis, including classification, regression, clustering, dimensionality reduction, model selection, and preprocessing. scikit-learn is known for its consistent API design, thorough documentation, and being the go-to library for classical machine learning in Python.

## Key Attributes

- **Type**: Python library (machine learning)
- **Created by**: David Cournapeau (2007); maintained by a community of contributors under the NumFOCUS fiscal sponsor
- **Notable for**: Clean, consistent estimator API; extensive documentation; battle-tested algorithms for production ML
- **License**: BSD-3-Clause
- **Other facts**: Implements algorithms including SVMs, random forests, gradient boosting, k-means, PCA, and many more; does not focus on deep learning (PyTorch/TensorFlow handle that); commonly used for tabular data problems

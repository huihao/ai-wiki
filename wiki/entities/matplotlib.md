# matplotlib

## Summary
matplotlib is the foundational plotting and visualization library for Python, widely used across data science, machine learning, and scientific computing. It provides a comprehensive API for creating static, animated, and interactive plots, charts, histograms, and other visualizations, and serves as the visualization backend for many higher-level libraries.

## Key Attributes
- **Type**: Data visualization library
- **Language**: Python
- **Notable for**: The de facto standard for Python plotting; inspired the ggplot-style and MATLAB-style interfaces
- **Created**: 2003 by John D. Hunter
- **License**: Matplotlib License (BSD-compatible)
- **Key interfaces**: pyplot (MATLAB-style), object-oriented API, third-party wrappers (Seaborn, Plotly)
- **Backend support**: Agg, TkAgg, Qt5Agg, and many others

# Hugging Face

## Summary

Hugging Face is an AI platform providing documentation and tools for Large Language Model development and deployment, including function calling capabilities through their HUGS (Hugging Face services) inference infrastructure.

## Key Attributes

- **Type**: AI platform, documentation provider
- **Focus**: Democratizing AI through open source and open science
- **Services**: HUGS (hosted inference), transformers library
- **Documentation**: Guides for function calling, inference, tools API
- **Mission**: Advance and democratize artificial intelligence
- **Inference**: InferenceClient for local and hosted model access
- **Tools**: JSON schema function definitions, Pydantic model support

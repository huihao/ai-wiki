# Vercel

## Summary

Vercel is a cloud platform for frontend developers, known for Next.js hosting and serverless deployments. It has expanded into AI infrastructure with Vercel AI SDK, Fluid compute, and Vercel Sandbox for agentic workloads. Vercel created the AI SDK as a unified TypeScript toolkit for building AI-powered applications.

## Key Attributes

- **Key products**: Vercel Platform, Next.js, Vercel AI SDK, Vercel Sandbox, Fluid compute
- **CEO**: Guillermo Rauch
- **CTO**: Malte Ubl
- **AI SDK Features**: Modern streaming, fallbacks, multi-model support
- **AI SDK Support**: React, Next.js, Vue, Svelte, Node.js

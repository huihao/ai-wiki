# AWS Strands

## Summary

AWS Strands Agents is an open-source agent SDK and runtime developed by Amazon Web Services. It provides a framework for building, deploying, and running AI agents with built-in support for tool use, memory, and multi-agent coordination. Strands integrates with the broader AWS ecosystem for scalable agent deployment.

## Key Attributes
- **Type**: agent SDK / framework
- **Notable for**: providing a production-ready agent runtime within the AWS ecosystem
- **Other facts**: open-source; designed for building agents that can use tools, maintain state, and coordinate across multiple agents; integrates with AWS services for compute, storage, and model access

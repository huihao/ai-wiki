# AutoGen

## Summary

AutoGen is an open-source framework by Microsoft for building multi-agent AI applications where multiple LLM-powered agents can collaborate, converse, and perform tasks together using customizable and conversable agents.

## Key Attributes

- Created by Microsoft Research
- Enables multi-agent conversations and collaboration
- Supports human-in-the-loop patterns
- Agents can be customized with specific roles and capabilities
- Used for complex task decomposition and problem-solving

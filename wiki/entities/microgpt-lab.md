# MicroGPT Lab

Interactive lab that visualizes how a MicroGPT tokenizes data, learns patterns, and generates names.

An educational visualization tool demonstrating how GPT models work at a fundamental level, showing the tokenization process, pattern learning, and text generation through an interactive interface.

## Features
- Tokenization visualization
- Pattern learning demonstration
- Name generation examples
- Interactive UI for understanding GPT fundamentals

# Streamlit

## Summary

Open-source Python framework for building interactive data and machine learning applications without frontend development. Acquired by Snowflake in 2022. Widely used in the AI/ML community for rapid prototyping, model demos, and internal tools.

## Key Attributes

- **Type**: Open-source Python framework
- **Parent company**: Snowflake (acquired 2022)
- **License**: Apache 2.0
- **Core paradigm**: Script-based; re-runs entire script on user interaction
- **Deployment**: Streamlit Community Cloud (free), Snowflake (enterprise), self-hosted
- **Adoption**: 90%+ of Fortune 50 companies
- **Key users**: Google X, Stitch Fix, Uber, Yelp, Insight Data Science

# Obsidian

## Summary
Obsidian is a powerful knowledge base and note-taking application that works on top of a local folder of Markdown files. It is popular among researchers, developers, and writers for its bidirectional linking, graph visualization, and plugin ecosystem, enabling users to build personal knowledge graphs from interconnected notes.

## Key Attributes
- **Type**: Note-taking / knowledge management application
- **Notable for**: Bidirectional wiki-links, local-first Markdown files, graph visualization, and extensive plugin ecosystem
- **Platform**: Desktop (Windows, macOS, Linux), mobile (iOS, Android)
- **Data format**: Plain Markdown files (`.md`) stored locally
- **Key features**: Backlinks, graph view, daily notes, canvas, community plugins, Obsidian Sync (paid)
- **License**: Free for personal use; paid for Sync and Publish features
- **Architecture**: Built on Electron; data stays on the user's device (local-first)

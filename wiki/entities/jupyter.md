# Jupyter

## Summary
Jupyter is an open-source project that provides interactive computing environments through notebooks, kernels, and related tools. Originally spun out of the IPython project, Jupyter has become the standard platform for interactive data science, machine learning research, and scientific computing, supporting over 40 programming languages via its kernel architecture.

## Key Attributes
- **Type**: Open-source project / computing platform
- **Notable for**: Establishing the interactive notebook paradigm that became ubiquitous in data science
- **Created**: 2014 (evolved from IPython Notebook, first released 2011)
- **Components**: Jupyter Notebook, JupyterLab, JupyterHub, JupyterLite, IPython kernel
- **License**: BSD
- **Website**: jupyter.org

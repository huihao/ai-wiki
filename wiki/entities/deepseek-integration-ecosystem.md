# DeepSeek Integration Ecosystem

## Summary

A comprehensive collection of software applications, tools, and platforms that integrate DeepSeek's large language model capabilities. The ecosystem includes chat applications, browser extensions, coding assistants, knowledge management systems, and agent frameworks, demonstrating DeepSeek's versatility across different domains.

## Key Attributes

- **Scope**: Over 100+ integrated applications and tools
- **Categories**: Chat clients, browser extensions, IDE integrations, knowledge bases, agent platforms, productivity tools
- **API**: Available through DeepSeek Open Platform (platform.deepseek.com)
- **Integration approach**: API-compatible with OpenAI format, enabling easy integration

## Integration Categories

### Chat Applications
- SwiftChat: Cross-platform React Native chat with real-time streaming
- 4EVERChat: Multi-model comparison platform with unified API
- DeepChat: Desktop chat application
- Chatbox: AI chat interface
- ChatGPT-Next-Web: Web-based chat application
- LibreChat: Open-source chat interface

### Browser Extensions
- 沉浸式翻译 (Immersive Translate): Translation extension
- 沉浸式导读 (Immersive Reading Guide): Reading assistance
- ChatGPT Box: Browser-based AI assistant
- 划词翻译 (Word Selection Translate): Translation tool
- 流畅阅读 (Fluent Read): Reading enhancement

- Continue: VS Code AI coding assistant
- Cline: Autonomous coding agent
- AI Commit: Git commit message generator
- SeekCode Copilot: Code completion tool

### Knowledge Management

- 思源笔记 (SiYuan Note): Note-taking application

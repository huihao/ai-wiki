# Grafana

## Summary
Grafana is an open-source analytics and interactive visualization platform that provides dashboards, graphs, and alerts for monitoring infrastructure and application metrics. It is a central component of modern observability stacks, commonly paired with Prometheus for metrics, Loki for logs, and Tempo for traces. Grafana Labs, the company behind the project, also develops Grafana Cloud, a fully managed observability platform with AI/ML-powered anomaly detection and forecasting capabilities.

## Key Attributes
- **Type**: observability and visualization platform (software)
- **Notable for**: de facto standard for metrics dashboards; open-source core with commercial cloud offering
- **Created by**: Torkel Ödegaard

- **Developer**: Grafana Labs
- **License**: AGPLv3 (open-source version)

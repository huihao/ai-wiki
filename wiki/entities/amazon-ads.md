# Amazon Ads

## Summary

Amazon Ads is Amazon's advertising platform, which uses PyTorch and TorchServe for large-scale ML model serving to power ad ranking, targeting, and personalization across Amazon's retail ecosystem.

## Key Attributes

- Uses TorchServe and AWS Inferentia for model serving at scale
- Powers ad ranking and personalization across Amazon's retail platforms
- Case study in scaling ML models with PyTorch infrastructure

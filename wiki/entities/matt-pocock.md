# Matt Pocock

## Summary

Matt Pocock is a TypeScript educator and creator of TotalTypeScript, known for making complex TypeScript concepts accessible. He authored the widely-cited "Getting Started With Ralph" guide that explains how to set up Ralph loops using Claude Code in Docker sandboxes, including the `ralph-once.sh` (human-in-the-loop) and `afk-ralph.sh` (autonomous) patterns.

## Key Attributes

- Creator of TotalTypeScript (TypeScript education platform)
- Creator of the Ralph technique for autonomous agent loops
- TypeScript educator and developer advocate

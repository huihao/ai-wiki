# Starlette

## Summary

Starlette is a lightweight ASGI framework/toolkit for building async Python web services, serving as the foundation for FastAPI and providing low-level utilities for routing, middleware, and request handling.

## Key Attributes

- Lightweight ASGI framework for Python
- Foundation of FastAPI web framework
- Supports async/await natively
- Provides routing, middleware, WebSocket, and static file serving
- Designed for high performance and simplicity

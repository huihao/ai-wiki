# Snyk

Snyk is a developer-focused security platform that helps developers find, fix, and monitor vulnerabilities across:
- Open source dependencies
- Container infrastructure
- Infrastructure as Code (IaC)
- Application code

## History

- **Founded**: 2015
- **Mission**: Developer-first security
- **Acquisition**: June 24, 2025 - Acquired Invariant Labs

## Products

### Snyk Open Source
Scans for vulnerabilities in npm, Maven, and other package registries.

### Snyk Container
Security scanning for container images and Kubernetes workloads.

### Snyk Infrastructure as Code
Detects security misconfigurations in Terraform, CloudFormation, Kubernetes YAML, and Ansible.

### Snyk Code
Static application security testing (SAST) for source code.

### Snyk Labs
Research division focused on:
- AI agent security
- Developer security tools
- Emerging security challenges

## Acquisition of Invariant Labs (June 2025)

Snyk acquired Invariant Labs to:
- Accelerate agentic AI security innovation
- Deepen Snyk Labs research capabilities
- Integrate agent security into developer workflows
- Address security challenges in autonomous AI systems

## Developer-First Approach

Snyk focuses on:
- Integrating security into developer workflows
- IDE extensions and CI/CD integrations
- Automated vulnerability detection
- Fix recommendations
- Security policy enforcement

## Security Coverage

- **Languages**: JavaScript, Python, Java, Go, Ruby, PHP, .NET, and more
- **Package Managers**: npm, Maven, Pip, Gradle, NuGet, and others
- **Infrastructure**: Docker, Kubernetes, Terraform, CloudFormation, Ansible
- **Cloud Providers**: AWS, Azure, Google Cloud

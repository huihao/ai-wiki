# fast.ai

## Summary

Educational organization and open-source deep learning library co-founded by [[jeremy-howard]] and [[rachel-thomas]]. Known for the free Practical Deep Learning for Coders course and the fastai library, which provides high-level APIs on top of PyTorch. Mission: make deep learning accessible to domain experts and practitioners without requiring advanced math or expensive hardware.

## Key Attributes

- **Type**: Non-profit educational organization + open-source software project
- **Products**:
  - fastai library — high-level PyTorch wrapper
  - Practical Deep Learning for Coders — free course
  - Deep Learning for Coders book (O'Reilly, free online)
- **Philosophy**: Top-down teaching; practical first, theory later; minimal prerequisites
- **Community**: Active forums at forums.fast.ai
- **Impact**: Alumni at Google Brain, OpenAI, Tesla, Adobe, Amazon; published research at NeurIPS

# Bengio

Yoshua Bengio是深度学习领域的先驱之一，蒙特利尔大学教授，图灵奖获得者。

## 主要贡献

### Seq2Seq概念

2014年论文"Learning Phrase Representations using RNN Encoder–Decoder for Statistical Machine Translation"提出了seq2seq（序列到序列）概念。

### 注意力机制

2015年论文"NEURAL MACHINE TRANSLATION BY JOINTLY LEARNING TO ALIGN AND TRANSLATE"提出注意力机制，解决了编码器-解码器模型的信息瓶颈问题。

# Jiayi Pan

## Summary

Jiayi Pan is a researcher and author of the paper "Learning Adaptive Parallel Reasoning with Language Models," which introduced the APR paradigm — the concept that reasoning models can learn to dynamically control their own parallelization strategy at inference time via special control tokens.

## Key Attributes

- **Known for**: Introducing the Adaptive Parallel Reasoning paradigm

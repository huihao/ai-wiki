# Spring AI

## Summary

Spring AI is an open-source framework by VMware/Broadcom for building AI-powered applications within the Spring ecosystem, providing abstractions for LLM interaction, RAG, and vector stores.

## Key Attributes

- Part of the Spring ecosystem for Java/JVM developers
- Provides abstractions for LLM providers (OpenAI, Azure, Ollama)
- Supports RAG with vector store integrations
- Simplifies AI integration in enterprise Java applications

# Haystack

## Summary
Haystack is an open-source NLP framework developed by deepset for building search, question-answering, and retrieval-augmented generation (RAG) systems using large language models. It provides a pipeline-based architecture with modular components for document retrieval, embedding, re-ranking, and text generation, allowing developers to compose custom LLM application workflows. Haystack integrates with vector databases, embedding models, and various LLM providers.

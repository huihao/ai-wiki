# Data Governance

## Summary
Data governance is the set of processes, policies, and standards that ensure the quality, security, privacy, and usability of an organization's data assets. It defines roles, responsibilities, and accountability for data management across the enterprise. Effective data governance is essential for regulatory compliance, data-driven decision-making, and maintaining trust in data systems.

## Key Attributes
- **Type**: concept/method
- **Notable for**: establishing organizational frameworks for managing data as a strategic asset
- **Other facts**: encompasses data quality management, metadata management, data lineage, access control, and compliance; key frameworks include DAMA-DMBOK and COBIT; increasingly important with regulations like GDPR, CCPA, and AI governance requirements; intersects with data engineering and ML operations for model training data

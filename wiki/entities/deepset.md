# deepset

## Summary

deepset is a company building open-source NLP and LLM tooling, best known for Haystack — a framework for building production-grade RAG pipelines and search systems. Founded in Berlin, Germany.

## Key Attributes

- Creator of Haystack 2.0, an open-source NLP/LLM pipeline framework
- Focuses on enterprise search, RAG, and information retrieval
- Supports integration with Elasticsearch, OpenAI, HuggingFace models
- Berlin-based AI company

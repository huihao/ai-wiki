# CrewAI

## Summary
CrewAI is an open-source framework for orchestrating multi-agent AI systems, enabling developers to create teams of AI agents that collaborate on complex tasks through role-based assignment. Each agent in a CrewAI "crew" is assigned a role, goal, and backstory, and agents work together sequentially or in parallel to accomplish objectives. It has become one of the more popular multi-agent orchestration frameworks, often compared with LangGraph and AutoGen.

- **Type**: open-source software framework

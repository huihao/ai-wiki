# Azure

## Summary
Microsoft Azure is a comprehensive cloud computing platform offering IaaS, PaaS, and SaaS services. Azure is a primary partner for OpenAI and provides Azure OpenAI Service, giving enterprises access to GPT-4 and other models with enterprise-grade security, compliance, and SLAs. Azure also offers Azure Machine Learning, Azure AI Search (formerly Cognitive Search), and a wide array of AI developer tools, making it one of the most AI-integrated cloud platforms.

## Key Attributes

- **Notable for**: Azure OpenAI Service partnership; deep enterprise AI integration
- **Launched**: 2010

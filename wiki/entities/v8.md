# V8

## Summary

V8 is Google's open-source JavaScript and WebAssembly engine, written in C++. It is used in Google Chrome and the Node.js runtime to execute JavaScript code. V8 compiles JavaScript directly to native machine code using just-in-time (JIT) compilation, enabling high-performance execution. It also supports WebAssembly for near-native-speed execution of code written in other languages.

## Key Attributes
- **Type**: JavaScript/WebAssembly engine
- **Notable for**: JIT compilation of JavaScript to machine code, powering both Chrome and Node.js
- **Other facts**: originally developed by Lars Bak and Kasper Lund at Google; first released in 2008; supports ECMAScript and WebAssembly specifications; uses hidden classes and inline caching for optimization

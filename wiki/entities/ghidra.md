# Ghidra

## Summary

Ghidra is a software reverse engineering (SRE) framework developed by the NSA, providing disassembly, decompilation, and analysis capabilities for analyzing compiled binaries without source code access.

## Key Attributes

- Developed by the NSA (National Security Agency)
- Open-source software reverse engineering tool
- Supports multiple processor architectures (x86, ARM, MIPS, etc.)
- Provides decompilation to C-like pseudocode
- Used in CTF challenges, malware analysis, and security research

# Composio

**Type**: Framework
**Category**: Tool Integration, API Management
**Organization**: Composio

## Description

Framework focused on simplifying integration of external tools and APIs into AI agents. Provides managed layer for authentication and execution of pre-built tools.

## Key Features

- **Universal Connector**: Single interface for many tools
- **Managed Authentication**: Handles API auth automatically
- **Pre-Built Tools**: 100+ ready-to-use integrations
- **Tool Wrappers**: No custom code needed
- **Easy Integration**: Quick capability addition

## Supported Tools/Services

- GitHub - Repository management
- Slack - Team communication
- Google Workspace - Docs, Sheets, Drive
- Notion - Knowledge management
- Airtable - Database operations
- Many others - 100+ integrations

## Integration with Gemini

- Uses Gemini's function calling
- Intelligent tool selection
- Automatic parameter mapping
- Context-aware tool usage
- Real-world task execution

## Advantages

- No individual API auth management

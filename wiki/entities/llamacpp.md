# llama.cpp

## Summary

llama.cpp is a C/C++ port of Meta's LLaMA model that enables efficient local inference of LLMs on consumer hardware (CPU, Apple Silicon, NVIDIA GPUs) without requiring Python or cloud APIs.

## Key Attributes

- C/C++ implementation of LLaMA inference
- Runs LLMs locally on CPU, Metal (Apple), CUDA (NVIDIA)
- Quantization support (GGUF format) for reduced memory usage
- No Python dependency required
- Supports various LLaMA-family models

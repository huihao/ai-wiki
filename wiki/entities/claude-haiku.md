# Claude Haiku

## Summary

Anthropic's fast, efficient model in the Claude family, optimized for speed and cost-effectiveness. Designed for high-volume, low-latency tasks where responsiveness matters more than maximum reasoning depth.

## Key Attributes

- **Developer**: Anthropic
- **Positioning**: Fastest and cheapest in the Claude family (Haiku < Sonnet < Opus)
- **Strengths**: Low latency, high throughput, cost-effective for large-scale usage
- **Use Cases**: Classification, content moderation, simple summarization, chatbots, data extraction, routing
- **API Access**: Available through Anthropic's Messages API with the same interface as other Claude models

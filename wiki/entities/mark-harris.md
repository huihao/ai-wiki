# Mark Harris

## Summary
Mark Harris is an NVIDIA developer and technical blogger known for authoring CUDA programming tutorials. He created accessible, hands-on GPU programming guides that progressively demonstrate optimization techniques, most notably "An Even Easier Introduction to CUDA," which walks through achieving a 2000x speedup through systematic GPU optimization.

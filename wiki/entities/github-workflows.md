# GitHub Actions / Workflows

## Summary

GitHub Actions is a CI/CD automation platform integrated into GitHub, using YAML workflow files to define automated build, test, and deployment pipelines triggered by repository events (push, PR, schedule, etc.).

## Key Attributes

- YAML-based workflow definitions in `.github/workflows/`
- Triggers: push, pull request, schedule, manual dispatch, workflow_dispatch
- Marketplace of reusable actions (build, test, deploy, security scan)
- Matrix builds for cross-platform/version testing
- Secrets management for credentials
- Reusable workflows and composite actions

# n8n

## Summary

Open-source workflow automation platform with a visual, node-based interface. Users connect services, APIs, and databases to automate tasks. n8n can be self-hosted or used as a cloud service. It competes with proprietary platforms like Zapier and Make, offering greater flexibility and data control.

## Key Attributes

- **Type**: Open-source workflow automation tool
- **License**: Fair-code (source-available, not OSI-approved open source)
- **Deployment**: Self-hosted or n8n Cloud
- **Interface**: Visual node-based editor + custom JavaScript/Python code nodes
- **Key strengths**: Extensive integrations (400+ nodes), API orchestration, data transformation, DevOps automation
- **AI capabilities**: AI nodes added in recent versions; can also integrate OpenAI/Anthropic via HTTP requests

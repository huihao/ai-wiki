# Modular

AI infrastructure company that created the Mojo programming language and the MAX platform for AI compute.

## Summary

Modular was founded by Chris Lattner (creator of LLVM, Swift, and MLIR) to build a unified AI compute platform. Their flagship product is Mojo, a systems programming language designed as a Python superset with C-level performance. The company aims to eliminate the fragmentation in ML infrastructure (CUDA, Triton, XLA) by providing a single, high-performance compute layer built on MLIR.

## Key Products

- **Mojo**: Python-superset language with systems programming capabilities (ownership, SIMD, GPU kernels)
- **MAX**: End-to-end AI inference and training platform
- **Mojo Manual**: Comprehensive language documentation

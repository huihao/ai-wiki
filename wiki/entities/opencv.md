# OpenCV

## Summary

OpenCV (Open Source Computer Vision Library) is an open-source library containing hundreds of computer vision algorithms, with a C++ API (OpenCV 2.x) featuring automatic memory management, modular structure, and cross-language bindings.

## Key Attributes

- **Type**: Open-source computer vision library
- **Version**: 4.14.0-pre (documented)
- **API**: C++ based (C API deprecated since 2.4)
- **Modules**: Multiple shared/static libraries
- **Namespace**: `cv::` for all classes/functions
- **Memory**: Automatic management with reference counting
- **Compatibility**: Python, Java, Matlab bindings
- **License**: Open source

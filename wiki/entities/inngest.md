# Inngest

## Summary

Inngest is a durable execution platform for building reliable, event-driven applications, providing step functions, scheduled workflows, and fan-out patterns for modern backend applications.

## Key Attributes

- Durable execution platform for event-driven apps
- Provides step functions with built-in retries and concurrency control
- Supports scheduled, cron-based, and event-triggered workflows
- Used in agent runtimes for durable, stateful agent execution
- Offers local development server and cloud-hosted platform

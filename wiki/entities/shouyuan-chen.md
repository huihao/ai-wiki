# Shouyuan Chen

## Summary

Shouyuan Chen is a Meta researcher who contributed to Position Interpolation (PI), a technique that scaled LLM context windows from 2K to 32K tokens by compressing position indices rather than extrapolating beyond training range. The core insight was that models memorize position encodings rather than learning position mathematics, so scaling positions to stay within the training range enables effective context extension.

## Key Attributes

- Meta researcher
- Contributor to Position Interpolation (PI) for context window extension
- Work enabled 2K to 32K context scaling through position compression

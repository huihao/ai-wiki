# Seunghyun SEO

## Overview

Researcher and blogger focused on neural network scaling, hyperparameter transfer, and training optimization. Author of comprehensive guide on scaling neural networks.

## Contributions

- Detailed guide on "How to Scale Neural Networks" covering muP (Maximal Update Parameterization)
- First-principles derivation of hyperparameter transfer rules
- Coverage of critical batch size, scaling laws, and practical examples from DeepSeek and StepFun

## Philosophy

- Focuses on understanding "how to scale" as: how to set initialization std, learning rate, batch size, and other hyperparameters as model size and dataset size grow
- Emphasizes that "more compute always leads to better performance" is only true with near-optimal hyperparameters

# Nemotron

## Summary

Nemotron is NVIDIA's family of large language models optimized for enterprise and agentic use cases. Notable models include Nemotron-3-Nano-4B, a compact 4B-parameter model that achieved 80.9% pass rate on Bash command generation tasks in NVIDIA AI Red Team research.

## Key Attributes

- **Developer**: [[nvidia]]
- **Notable model**: Nemotron-3-Nano-4B
- **Strengths**: Strong performance on agentic coding tasks despite small size
- **Benchmark**: 88.3% pass rate with grammar-constrained decoding (NVIDIA Bash tasks)

# Microsoft Defender
## Summary
Microsoft's built-in antimalware solution for Windows operating systems. Uses signature-based, heuristic, and behavioral detection mechanisms. Despite being a common detection target in malware development research, it consistently detects even basic shellcode loaders.
## Key Attributes
- **Type**: Antivirus/antimalware software
- **Developer**: Microsoft
- **Detection methods**: Signature, heuristic, behavioral, cloud-based

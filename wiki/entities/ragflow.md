# RAGFlow

## Summary

RAGFlow is an open-source engine employing deep document understanding as the driving force for building optimized RAG applications. It integrates both structured and unstructured data for improved question-answering tasks.

## Key Attributes

- **Type**: Open-source engine
- **Focus**: Deep document understanding for RAG
- **Capabilities**: Handles structured and unstructured data, citation-grounded Q&A
- **Format Support**: PDF files, images, multiple document formats
- **Architecture**: Scalable and modular
- **Use Case**: RAG systems adapted to specific user needs

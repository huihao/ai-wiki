# GitHub CLI (gh)

## Metadata

- **Type**: CLI Tool
- **Platform**: GitHub
- **URL**: https://cli.github.com
- **Purpose**: GitHub operations from terminal

## Definition

GitHub CLI (gh) is the official command-line tool for GitHub that enables users to perform GitHub operations (PRs, issues, releases, repo management) directly from the terminal without using the web interface.

## Key Features

- Pull request management.
- Issue tracking.
- Repository operations.
- Release management.
- Workflow automation.
- GitHub Actions integration.

## Common Commands

```bash
gh pr create
gh pr list
gh pr merge
gh issue create
gh repo create
gh release create
```

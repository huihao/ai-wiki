# nbdev

## Overview

A development framework from fast.ai that enables writing, testing, documenting, and distributing software packages entirely within Jupyter Notebooks. Eliminates context-switching between code editors, documentation tools, and test runners.

## Key Features

- **Write prose, code, and tests in notebooks** — no context-switching between tools
- **Beautiful documentation** — generates technical docs and scientific articles with Quarto
- **CI/CD out of the box** — GitHub Actions integration for testing and deployment
- **Publish to PyPI and conda** — distribute packages directly from notebooks
- **Git-friendly notebooks** — human-readable merge conflicts, no unwanted metadata
- **Two-way IDE sync** — work in notebooks or your favorite IDE

## Philosophy

nbdev embodies the fast.ai approach of "delightful software development" — reducing friction between exploration (notebooks) and production (packaged software). It treats notebooks as first-class source files rather than throwaway experiments.

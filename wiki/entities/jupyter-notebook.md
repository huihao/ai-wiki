# Jupyter Notebook

## Summary
Jupyter Notebook is an open-source web application that allows users to create and share documents containing live code, equations, visualizations, and narrative text. It is the primary file format and interface for the Jupyter ecosystem, widely used in data science, machine learning, and scientific computing for exploratory analysis and reproducible research.

## Key Attributes
- **Type**: Interactive computing application / file format
- **File extension**: `.ipynb` (JSON-based format)
- **Notable for**: Pioneering the literate programming approach for data science; mixing code, output, and documentation
- **Kernel support**: Python, R, Julia, and 40+ other language kernels
- **Key feature**: Cell-based execution model allowing iterative development with inline output

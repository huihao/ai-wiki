# Genkit

## Summary

Genkit is an open-source framework built by Google for building AI-powered full-stack applications. It provides unified APIs for multiple AI providers with built-in developer tools, structured outputs, and production monitoring.

## Key Attributes

- **Type**: AI Framework
- **Creator**: Google/Firebase
- **Languages**: JavaScript/TypeScript, Go, Python, Dart
- **Focus**: Full-stack AI application development
- **Features**: Unified API, local dev UI, deployment flexibility, monitoring

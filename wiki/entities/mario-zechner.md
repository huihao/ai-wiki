# Mario Zechner

## Summary

Mario Zechner is an independent developer and creator of Pi, a minimal coding agent harness. He advocates for extreme minimalism in agent design: tiny system prompts, only 4 core tools, no MCP, no built-in plan mode, and YOLO-by-default execution. Pi outperforms Claude Code and Codex on Terminal-Bench 2.0 benchmarks.

## Key Attributes

- **Known for**: Creator of Pi (minimal coding agent)
- **Philosophy**: "If I don't need it, it won't be built"
- **Other projects**: Sitegeist (browser-use agent), mom (agent framework)
- **Location**: Austria
- **Online**: mariozechner.at

# Semantic Kernel

## Summary

Semantic Kernel is an open-source AI orchestration SDK developed by Microsoft that enables developers to integrate large language models (LLMs) with conventional programming languages and application code. It provides abstractions for prompts, memory, plugins, and planners, allowing developers to build AI-enhanced applications with C#, Python, or Java. Semantic Kernel serves as the AI orchestration layer for Microsoft's developer ecosystem, including integration with Azure AI services.

## Key Attributes

- **Type**: AI orchestration SDK
- **Created by**: Microsoft (2023)
- **Notable for**: Multi-language support (C#, Python, Java); integration with Azure OpenAI Service; plugin architecture for extending LLM capabilities with native code
- **License**: MIT
- **Other facts**: Provides connectors to OpenAI, Azure OpenAI, Hugging Face, and other model providers; supports function calling, memory plugins, and planning patterns; closely integrated with Microsoft's broader AI platform including Azure AI Search and Microsoft 365 Copilot

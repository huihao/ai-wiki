# Volcano Engine (火山引擎)

## Summary

Volcano Engine is a cloud computing and AI platform provider, part of ByteDance, offering AgentKit solution for enterprise agent deployment and tool management.

## Key Attributes

- **Type**: Cloud platform company
- **Parent**: ByteDance
- **Focus**: Enterprise AI solutions, Agent tools
- **Products**: AgentKit Gateway, Skills management platform
- **Location**: China

## Key Products

### AgentKit Gateway

Traffic, control, and data hub for agent ecosystems:
- **Traffic Hub**: Unified processing of agent interactions and MCP calls
- **Control Hub**: MCP routing, model routing, load balancing
- **Data Hub**: Full lifecycle metadata management
- **Security**: Zero-trust identity and access management

### Skills Platform

Enterprise-level skills management:
- Generation from SOPs, templates, scripts
- Centralized management with version control
- Discovery and execution in sandboxed environment

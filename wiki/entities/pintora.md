# Pintora

## Summary

Pintora is a niche text-to-diagram language, less popular than Mermaid or PlantUML. Supports multiple diagram types: Sequence, ER, Component, Activity, Mindmap, Gantt, Class diagrams.

## Key Attributes

- **Type**: Text-to-diagram language
- **Popularity**: Niche (not widely adopted)
- **Diagram types**: Sequence, ER, Component, Activity, Mindmap, Gantt, Class
- **Website**: https://pintorajs.vercel.app/
- **CLI tool**: `@pintora/cli` for rendering diagrams

# OpenAI Codex

## Summary
OpenAI Codex is OpenAI's coding agent platform — not a single model but a product and workflow layer wrapping frontier models with file access, shell execution, sandboxes, approval flows, and code review. Originally derived from GPT-3 fine-tuned on code, the Codex line has evolved to include four surfaces (CLI, IDE extension, Codex app, Codex Cloud) and now supports GPT-5.5 as the flagship model (April 2026).

## Key Attributes
- **Type**: Coding agent platform
- **Developed by**: [[openai]]
- **Notable for**: Four surfaces (CLI, IDE, App, Cloud), sandboxed execution, code review, worktree support
- **Key models**: GPT-5.5 (flagship, April 2026), GPT-5.4, GPT-5.3-Codex, codex-mini
- **Surfaces**: CLI (terminal), IDE extension (VS Code/Cursor), Codex app (parallel tasks), Codex Cloud (background GitHub tasks)
- **Pricing**: Included with ChatGPT Plus/Pro/Business/Enterprise plans

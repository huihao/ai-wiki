# ChromaDB

## Summary

ChromaDB is an open-source vector database designed for AI applications, particularly RAG systems. It stores document embeddings and enables fast semantic search to retrieve relevant text chunks. ChromaDB integrates with LangChain and supports various embedding models for building retrievalaugmented generation pipelines.

## Key Attributes

- **Type**: Vector database / embedding store
- **Use case**: RAG applications, semantic search, document retrieval
- **Integration**: LangChain, Ollama, various embedding models
- **Features**: Persistent storage, fast retrieval, collection management

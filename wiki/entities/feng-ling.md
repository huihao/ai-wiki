# 凤聆 (Feng Ling)

## Summary

Alibaba Cloud Developer and author who built a fully automated [[automated-evaluation-platform|evaluation platform]] demonstrating AI-driven overnight system optimization. The platform enables AI agents to autonomously generate test sets, run evaluations (including browser-based UI testing), submit reports, and iteratively improve target systems — with zero human labor after initial task description.

# Groq

## Summary
Groq is an AI hardware and infrastructure company known for its Language Processing Unit (LPU), a custom inference chip designed for extremely low-latency language model serving. Unlike GPU-based approaches, Groq's LPU uses a deterministic, single-core streaming architecture that delivers consistent, high-speed token generation. Groq offers cloud-based API access to hosted models (including Meta LLaMA and Mixtral) with some of the fastest inference speeds available.

## Key Attributes
- **Type**: AI hardware and inference infrastructure company
- **Notable for**: LPU chip for ultra-low-latency LLM inference; hosted inference API
- **Founded**: 2016
- **Headquarters**: Mountain View, California, USA
- **Founder**: Jonathan Ross (former Google TPU engineer)
- **Key product**: Groq LPU, GroqCloud API

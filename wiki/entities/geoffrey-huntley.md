# Geoffrey Huntley

## Summary

Geoffrey Huntley is a developer who created the Ralph Wiggum Technique (Ralph Loop), a simple but powerful method for running AI coding agents autonomously. The agent reads a PRD and progress file, implements one task per iteration, commits, and updates progress — enabling humans to go AFK while the agent works.

## Key Attributes

- **Known for**: Ralph Wiggum Technique / Ralph Loop
- **Website**: ghuntley.com
- **Technique**: Simple bash loop with PRD + progress tracking

# Google Colab

## Summary

Google Colab (Colaboratory) is a free cloud-based Jupyter notebook environment that provides free access to GPU/TPU compute, making it a popular platform for ML education, prototyping, and experimentation.

## Key Attributes

- Free Jupyter notebook environment in the browser
- Free GPU and TPU access (with usage limits)
- Pre-installed ML libraries (TensorFlow, PyTorch, JAX)
- Integration with Google Drive for file storage
- Used extensively in ML courses and tutorials
- Pro and Enterprise tiers for more compute and longer runtimes

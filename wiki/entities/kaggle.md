# Kaggle

## Summary

Kaggle is Google's platform for data science competitions, notebooks, datasets, and community, providing free GPU/TPU compute and a marketplace for ML models and datasets.

## Key Attributes

- Owned by Google
- Hosts data science competitions and ML challenges
- Provides free Jupyter notebooks with GPU/TPU access
- Marketplace for datasets, models, and code
- Large data science community with 15M+ users

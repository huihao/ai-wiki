# Anki

## Summary

Anki is a free, open-source spaced repetition flashcard program widely used for memorization and long-term learning. It implements the SM-2 algorithm (and newer FSRS) to schedule card reviews at optimal intervals. Anki is particularly popular among medical students, language learners, and increasingly among machine learning practitioners who use it to retain technical concepts, formulas, and paper details.

## Key Attributes

- **Type**: Open-source spaced repetition software
- **Platforms**: Desktop (Windows/Mac/Linux), mobile (AnkiMobile iOS, AnkiDroid Android)
- **Key algorithm**: SM-2 (default), FSRS (optional newer scheduler)
- **Notable use case**: ML education — platforms like aiquizzes export to Anki decks

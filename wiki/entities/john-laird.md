# John Laird

## Summary

John Laird is a computer scientist at the University of Michigan and co-creator of the SOAR cognitive architecture (1987) with Allen Newell and Paul Rosenbloom. SOAR unified problem-solving, learning, and memory into a single framework, distinguishing between working memory for immediate tasks and long-term procedural memory for learned rules.

# Conda

## Summary

Conda is an open-source package management and environment management system for installing and managing software dependencies, particularly popular in the Python data science and ML ecosystem.

## Key Attributes

- Cross-platform package and environment manager
- Manages Python, R, and other language dependencies
- Creates isolated environments for different projects
- Bundled with Miniconda and Anaconda distributions
- Core tool for ML/DL project setup

# Roboflow

## Summary

Roboflow is a computer vision platform providing tools for dataset management, annotation, model training, and deployment. Roboflow Universe is the world's largest collection of open-source CV datasets and models.

## Key Attributes

- **Products**: Universe (datasets/models), Annotate, Train, Deploy
- **Universe scale**: 1B+ images, 1M+ datasets, 250K+ models
- **Tasks**: Object detection, instance segmentation, classification

# Node.js

## Summary

Node.js is an open-source JavaScript runtime built on Chrome's V8 engine, enabling server-side JavaScript execution with an event-driven, non-blocking I/O model designed for scalable network applications.

## Key Attributes

- Built on V8 JavaScript engine (Chrome's JIT-compiled JS engine)
- Event-driven, non-blocking I/O model
- npm: largest software package registry (2M+ packages)
- Native C++ addon support via N-API
- Built-in test runner (Node.js 18+)
- ES modules and CommonJS module systems
- Streams, File System, HTTP/HTTPS, WebSocket APIs

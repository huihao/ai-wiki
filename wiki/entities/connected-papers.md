# Connected Papers

## Summary

Connected Papers is a tool for building literature similarity graphs from a seed paper, visualizing citation networks, and discovering related work in a visual interface.

## Key Attributes

- **Type**: Literature visualization tool
- **Input**: Single "anchor paper" as seed
- **Output**: Similarity graph with related papers
- **Expansion**: 2 layers from anchor (method branches, alternatives)
- **Metadata**: Dataset, metrics, code availability per node

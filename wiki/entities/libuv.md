# libuv

## Summary

libuv is a cross-platform C library for asynchronous I/O, providing the event loop, networking, file system, and process management that powers Node.js's non-blocking I/O model.

## Key Attributes

- Cross-platform async I/O library (Linux, macOS, Windows)
- Core runtime library powering Node.js event loop
- Provides TCP/UDP networking, file system operations, and process management
- Written in C, with bindings for multiple languages
- Originally developed for Node.js, now used independently
